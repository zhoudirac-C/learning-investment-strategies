# AI资本开支周期与泡沫风险

> 来源：2026-05-23 视频《2028，那场必来的危机，AI资本狂潮的终局推演》。
> 本页记录 UP 的长期风险框架。涉及 2026-2028 时间线、国产 GPU 出货、EUV 商业化等内容均为待验证推演，不当作已确认事实。

## 核心定性

UP 将 AI 产业当前阶段定义为：

```
高资本开支 -> 短寿命资本品 -> 金融杠杆化 -> ROI压力上升 -> 潜在资本开支拐点
```

它和短线交易框架不是同一层级。短线仍看指数、量能、主线强度、承接和订单兑现；长期则要防止把 AI 硬件链的阶段性高景气误认为永久增长。

## 2026-05-24：美债收益率与6月黄金坑

5月24日周复盘把长期 AI 资本开支风险压缩到更近的交易窗口：30年期美债收益率已经在5%附近波动，真正需要警惕的区间是5.2%-5.5%。

UP 对本轮长端利率高企给出三条结构性解释：

| 原因 | 对AI交易的含义 |
|------|----------------|
| AI资本开支上修 | AI是重资产周期，持续需要数据中心、服务器、电力和网络投资 |
| 全球安全建厂 | 供应链从效率优先转向安全优先，本土化和冗余投资抬高资金需求 |
| 美国财政赤字 | 长端利率受财政供需约束，美联储不完全掌控定价 |

交易含义：美债收益率上行是流动性冲击，不一定立刻刺破AI泡沫，但可能制造6月深度回调。若回调兑现，UP将其视为参与后续AI泡沫加速阶段的“黄金坑”。

这条判断目前仍是短期情景推演。使用时不能机械地把“6月回调”当作必然结论，而应跟踪：

- 30年期美债收益率是否进入5.2%-5.5%；
- 美联储6月会议是否释放鹰派冲击；
- 油价下跌是否缓解通胀预期；
- AI硬件核心股是否出现拥挤交易松动；
- A股是否在周一周二反弹后于周四周五转弱。

## 三条风险链

### 1. 资本开支流量链

AI资本开支不仅影响英伟达和服务器厂商，也影响数据中心建设、电力设备、工程施工、云服务、金融融资和就业。若云厂商或AI公司资本开支明显下修，冲击可能从订单收缩传导到就业、消费、信心和债务压力。

对交易的启发：

- AI服务器、PCB、MLCC、ABF、存储、电源液冷等方向，不能只看BOM价值量提升；
- 还要跟踪云厂商资本开支、AI公司收入增速、订单能见度和实际ROI；
- 资本开支周期未变坏前，不把长期危机推演当作日内卖点。

### 2. GPU短寿命抵押品链

UP 把 GPU 定义为短寿命资本品：物理寿命可能较长，但经济寿命受技术迭代影响更短。若GPU租赁公司、云厂商或基金把GPU作为抵押品加杠杆，二手价格下跌会造成抵押品贬值、补保证金、被迫抛售和融资条件恶化的负反馈。

重点跟踪：

- GPU利用率；
- 算力租金价格；
- 折旧和资产减值；
- GPU二手价；
- GPU租赁公司债券利差和再融资能力。

### 3. 摩尔定律衰减与ROI链

VR200机柜价格大幅提升，一方面证明AI硬件价值量扩散，利好存储、PCB、MLCC、ABF、电源液冷等环节；另一方面也说明性能提升越来越依赖堆料，客户ROI压力提高。

因此，Rubin BOM涨价有双重含义：

| 含义 | 交易解释 |
|------|----------|
| 景气信号 | 硬件组件价值量提升，带来产业链扩散机会 |
| 风险信号 | 单机柜成本上升，客户更关注回本周期和资本开支效率 |

## UP给出的长期时间线

| 阶段 | UP推演 | 学习处理 |
|------|--------|----------|
| 2026 Q3-Q4 | 大模型收入增速下滑、资本开支指引下调 | 作为观察假设 |
| 2026年底-2027 Q1 | GPU二手价和HBM供需出现拐点 | 作为风险扳机观察 |
| 2027 Q2-Q3 | GPU租赁公司债务压力显现 | 观察融资利差和利用率 |
| 2027 Q4-2028 Q1 | 头部租赁公司违约或破产，形成AI版雷曼时刻 | 低置信度情景推演 |
| 2028年后 | 算力价格下降，AI进入普及展开期 | 作为技术革命周期框架 |

这条时间线不是当前交易结论。只有当多个监控信号同时恶化，才上修为系统性风险。

| 信号 | 风险含义 |
|------|----------|
| 台积电先进制程产能利用率跌破90% | 先进AI芯片需求可能降温 |
| HBM现货价/合约价跌破0.85 | 存储供需从紧缺转向过剩 |

| 云厂商资本开支增速降到15%以下 | AI基础设施扩张降速 |
| GPU租赁公司债券利差超过500bps | 金融系统开始重估抵押品风险 |
| H100二手价跌破1.8万美元 | GPU残值预期恶化 |
| 中国第二代EUV进入小批量验证 | 全球算力硬件供给格局可能变化 |

其中部分阈值来自UP推演，需后续结合外部数据验证。

## 中国AI普及阶段假设

UP认为，泡沫破裂后算力降价，AI从高价精英服务进入普及阶段，中国可能因以下优势受益：

- 低成本、开源和普惠式模型打法；
- 电力基建和制造链完整；
- 国产GPU、国产算力和半导体替代持续推进；
- 发展中国家更需要便宜、够用的AI服务。

这个判断可以作为国产算力、国产CPU/GPU、AI应用和电力基础设施的长期背景，但仍需用出货、客户、生态、现金流和财报验证。

## 对当前交易的用法

1. **不提前悲观**：长期危机框架不等于当前AI硬件主线立刻结束。
2. **不线性外推**：BOM涨价和订单高景气不能无条件外推为永久增长。
3. **加一层压力测试**：分析AI硬件链时，除了个股K线和订单，还要看资本开支、GPU二手价、HBM价格和融资条件。
4. **区分技术与资产价格**：技术长期价值可能继续提升，资产价格仍可能因估值和杠杆出清而大幅波动。
5. **个人先保现金流**：参与高波动科技周期前，先确保生活和投资现金流不断裂。
