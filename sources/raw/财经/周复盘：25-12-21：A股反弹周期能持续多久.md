## 一、技术面：反弹周期与节奏

### 当前判断

- 上周三 60 分钟级别出现明显底部信号，90 分钟、120 分钟均有 MACD 金叉 + 红角线
- 对应**反弹周期约 12 个交易日**，差不多到 1 月初
- 上周三底部形态确定 → 可直接满仓（博主已提示）
- 明天（周一）上证再涨 0.6% 就接近前期压力位，叠加周五利空消息 → **明天跌的概率较大**
- **不用担心**，如果跌了就是很好的加仓位置

### 策略

- 目前是磨底阶段，没有明显主线行情
- 没有主线 → 做**指数型 ETF**（宽基），指数涨多久你涨多久
- 等主线出来再切换到主线
- 不追高开票，等冲高回调两三天、借大盘低点再买入

## 三、2026 年方向定调：科技 > 消费

### 政策取向

- 中央经济工作会议：消费不是明年重点，**科技才是**
- 消费只要能拖着就行
- 不通过强刺激解决通缩，而是用更健康的手段——通缩周期会延长，但这是正确做法

### 明年核心方向

- 高端制造、半导体、国产替代、电力、军工
- 高股息能源国企（通缩 + 低利率环境下的避风港）

### 消费的策略性参与

- 不是不能看，但只能在特定窗口参与
- PSL（抵押补充贷款）等定向资金会用于服务型消费 → 出消息时有一波行情
- 消费走不远，没有政策刺激就没办法做
- 策略：科技行情走完一波 → 消费低迷 → 低位参与消费 → 等政策配合大盘
- 如果税费减免等政策力度不够 → 对消费保持警惕

### 为什么明年必须抓出口

- GDP 要达到 5% 或 5.5%，靠的是**出口**而非消费
- 出口 = 托底 + 去产能 / 反内卷 + 地缘博弈筹码
- 强拉消费 = 长期性政策 → 财政负担极重，效果不一定好
- 内需偏弱 + 通缩压力持续 → 多数企业盈利很差 → 很难走出戴维斯双击
- 消费类大概率不是明年戴维斯双击的方向

## 四、科技行业的本质变化

### 半导体从「渣男」到戴维斯双击

- 以前半导体被称为渣男行业，连从业人员都不信能起来
- 上一波牛市半导体稍微起来一点，整个熊市表现极差
- 但今年半导体**真正出业绩了** → 这一轮是戴维斯双击行情
- 对比美国：特朗普政策太多；我们反而更像在 all-in 科技
- **科技首次被重点写入十五五规划** → 持续性的政策资金可期

### 反内卷的本质

- 反内卷不是目的，是手段
- 让科技公司能赚钱、能有定价权
- 没有利润的科技公司很难活下去，不能一直靠国家补贴
- 反内卷 = 帮助科技企业活下来

### 沐曦股份、摩尔线程等

- 本周推出新产品，摩尔线程性能翻了 15 倍
- 这些都是明确的变化

### AI 应用的期待

- 今年 DeepSeek 行情被特朗普贸易战中断
- DeepSeek 的二股东（每日互动）当时走了一波浩浩荡荡的行情，如果没有贸易战甚至能走二波三波
- 明年能否迎来更多 DeepSeek 时刻？可以期待
- 阿里近期动作频繁：卫宁健康（AI 医疗）、AI 眼镜等
- 关键：这些动作能否转化为**业绩**？
- 如果之前不盈利的科技公司明年出现**业绩反转** → 戴维斯双击

## 五、人民币：明年大概率走强

- 目前人民币表现还行但购买力水平低 → 实际上是弱人民币
- 弱人民币有好处（稳出口），但现在不需要——我们出口走强也不需要弱人民币去稳，房地产也不需要兜底
- **明年人民币走强是大概率事件**
- 高盛：人民币至少升值 25%，有的经济学家直接喊升值 50%，甚至看到 **6.25**
- 需要研究日本当年的经验（广场协议），博主计划出专题

### 人民币升值的意义

- 信号发出 → 看科技类哪些能走戴维斯双击 → 果断参与
- 一旦走出来 → 一定是主线行情

## 六、如何识别主线行情

### 主线 vs 非主线的判断标准

| 维度 | 判断方法 |
|------|----------|
| **消息大小** | 最大级别 = 政策。是否上升到国家战略 / 大国博弈层面？ |
| **海外锚点** | 有没有海外对标股走出浩浩荡荡的行情？ |
| **持续性** | 会不会今天进明天套？主线不会一天两天结束，会给你足够时间研究 |
| **发散性** | 有没有扩散到多个子题材？稳定币 → RWA；商业航天 → 太空算力 / 火箭发射 / 卫星制造 |
| **资金共振** | 游资打第一波，机构后续介入，散户、外资都愿意参与 |

### 商业航天的完整演绎

| 阶段 | 特征 | 代表标的 |
|------|------|----------|
| **游资启动** | 消息催化前就启动，连板打高度 | 航天发展、航发动力 |
| **机构介入** | 12 月初中国卫星开始走趋势（机构票） | 中国卫星 |
| **分化** | 上周三大盘走强 → 游资行情结束，机构主导 | — |
| **后续** | 机构建完仓后在里面做 T，消息催化时散户 + 游资再冲一波 | 反复活跃，但节奏不同 |

- 机构票的特征：上影线多（涨了机构就砸，天天做 T）
- 业绩披露期前：确定有业绩的方向会再迎一波炒作
- 每年大主线很多，熊市每年也有三四波，现在更多 → 不用焦虑踏空

## 七、短线交易的核心纪律

### 固定模式是第一要务

- 短线内风格极多：首板、半路板、龙头、龙回头……
- 不能什么都想做 → **必须固定一种模式**
- 固定模式 = 固定买卖点 → 不会随手交易 → 才能稳定盈利
- 卖飞是常态，有固定买卖点就不存在「不敢买」或「卖飞」

### 小资金阶段

- 新手资金体量低 → 固定一种模式严格执行
- 稳定盈利后再去扩展风格
- 张盟主天天上龙虎榜，但那几个亿只占他仓位一两个点；更大的仓位在寒武纪这种容量票里

### 强度判断

| 指标 | 强 | 弱 |
|------|-----|-----|
| 封单额 | 成交几千万，封单几个亿 | 成交十几亿，封单一般 |
| 封板时间 | 开盘十分钟内封板 | 尾盘勉强封板 |
| 成交缩量 | 封板后缩量，无烂板 | 反复开板放量 |

### 盯盘核心：短线精灵

- 通达信「主力监控精灵」/ 其他软件的「短线精灵」
- 同一时间集体异动 → 板块启动 → 率先涨停的是龙头 → 龙头买不进就做龙二
- 你做几年之后，看到异动名字就知道它是哪个板块的 → 这才叫入门
- 龙头打到四板五板以上 → 脱离板块，属「昨日连板」/「妖股」板块 → 跟着整体短线情绪周期走

## 九、操作层面的补充

### 指数见底但没有主线时

- 买宽基 ETF，指数涨多久你涨多久
- 不要在拐点去追已经涨疯的方向

### 拐点出来的新方向

- 上周三指数反转 → 浙江世宝这种时机出来的票 = 天时地利人和
- 如果有持续消息推动 → 非常有看头

### 防守板块的跷跷板

- 科技 vs 消费（医药、食品）是跷跷板
- 大盘不好 → 做一两天防守；行情好 → 做科技
- 小的品种构不成跷跷板，不用看那么细

## 十、博主的角色定位

> 你把我招进你的团队，我是你的研究员和风控。你的老板，你做决策，你对你的钱负责。

- 团队分工：你是基金经理（管钱 + 决策），博主 = 研究员 + 风控
- 一个人既做基金经理又做风控 = 最危险（证监会都不会允许）
- 博主的职责：告诉你什么时候该谨慎，什么时候可以重仓
- 最终目标：教会你自己独立判断风险，即使未来没有博主也能做好
- **市场让你赚钱，不是博主让你赚钱。博主只是工具。**
