# F10财务分析（四）之PEG成长估值法

## 一、PEG 解决什么问题

**场景假设：**

| | 半导体成长股 | 消费股 |
|------|------------|--------|
| PE | 55 倍 | 18 倍 |
| 净利润增速 | **85%** | 3% |

纯看 PE，消费股便宜、半导体贵。带入增速后，结论完全反转。**PEG 在 PE 基础上加入成长速度维度，让估值更接近真实的贵与便宜。**

---

## 二、PEG 公式

```
PEG = PE ÷ G

PE = 市盈率
G  = 净利润增速（百分比去掉百分号，如 40% 写 40，不是 0.4）
```

**示例：**

| 公司 | PE | 增速 G | PEG | 判断 |
|------|-----|--------|-----|------|
| A | 40 | 40% | 40÷40 = **1.0** | 合理估值 |
| B | 20 | 5% | 20÷5 = **4.0** | 严重高估 |

**核心逻辑：PE 高不一定贵，PE 低不一定便宜。贵和便宜必须对应增速来看。**

---

## 三、戴维斯双击：PEG 的底层机制

**戴维斯双击 = 业绩增长 × 估值扩张 的双乘数效应**

当一家成长股业绩增速从 10% 提速到 30%：
1. **利润增长** → EPS 提升（业绩贡献）
2. **市场重估** → 愿意给更高 PE 倍数（估值扩张）
3. 两者叠加 → 股价几何级上涨

**典型路径：**
```
业绩翻倍 + PE 从 20→40 → 股价 ×4
业绩再翻倍 + PE 从 40→60 → 股价 ×3
两轮下来 = 12 倍
```

PEG 就是判断戴维斯双击过程中何时介入的核心工具。

## 四、EPS 计算：未来十二个月滚动法

### 关键：用未来一年 EPS，不是当年全年 EPS

**滚动 EPS 公式：**

```
未来一年 EPS = (今年剩余月数 ÷ 12) × 今年EPS + (明年已过月数 ÷ 12) × 明年EPS
```

**按月举例：**

| 当前月份 | 公式 |
|----------|------|
| 二月 | 10/12 × 今年EPS + 2/12 × 明年EPS |
| 三月 | 9/12 × 今年EPS + 3/12 × 明年EPS |
| 十月 | 2/12 × 今年EPS + 10/12 × 明年EPS |

### 明年 EPS 怎么来

- 取多家券商的一致预期（至少五家以上取均值）
- 可用大模型汇总各券商研报的预测值
- 不同券商之间差异可能超 30%，单家用偏差大

## 五、量能：PEG 的底层参数

**量能决定 PEG 的买卖标准，PEG 标准不是固定的，随市场流动性动态调整。**

| 量能环境 | PEG 安全买点 | PEG 危险区域 |
|----------|------------|------------|
| 缩量（~1.5 万亿） | < 0.7 | > 1.5 |
| 充沛（~2.5 万亿） | < 1.2 | > 3.0 |

**原则：** 市场水越多、钱越多，PEG 容忍度越高。永远先看量能，再看 PEG。不看量能就用 PEG 是盲目的。

## 六、实操演算：中际旭创（2026 年 2 月）

| 参数 | 数值 |
|------|------|
| 股价 | 531 元 |
| 2026 年 EPS | 18.11 |
| 2027 年 EPS（预期） | 25.22 |
| 当前量能 | 2 万亿 |
| 当前月份 | 2 月（剩余 10 个月） |

**第一步：滚动 EPS**
```
10/12 × 18.11 + 2/12 × 25.22 = 18.7
```

**第二步：动态 PE**
```
531 ÷ 18.7 = 28.4 倍
```

**第三步：成长率 G**
```
(25.22 - 18.11) ÷ 18.11 × 100 = 39.26
```

**第四步：PEG**
```
28.4 ÷ 39.26 = 0.72
```

**结论：** 0.72 在当前量能（2 万亿）下属有性价比的安全买点。股价越低，PEG 越低，买点越好。

## 七、案例对比：寒武纪的 PEG 变迁

| 时间 | PE | 增速 | PEG | 判断 |
|------|-----|------|-----|------|
| 2025 年初 | 80-100 倍 | 爆发增长 | < 1 | 合理，机构可布局 |
| 2025 年中 | 150-200 倍 | 高增速 | 1.2-2 | 需谨慎 |
| 2025 年 8 月高点 | **270 倍** | — | **> 3** | 情绪泡沫区 |

**关键数据：** 2025 年寒武纪净利润 16 亿，同比增速 2386%，扭亏为盈。字节跳动单季采购 6 万片（贡献约 6 亿收入）。

**核心启示：** 同一家公司，不同时间节点的 PEG 完全不同。年初是合理布局区，八月是情绪泡沫区。PEG 不区分公司好坏，区分的是当前价格相对于增速是高估还是低估。

## 八、适用范围

### ✅ 适用（有研发、有增速、增速可预测的成长型行业）

| 行业 |
|------|
| AI 算力链、AIDC、半导体设备 |
| 创新药、医疗器械、CXO |
| 高端制造、军工 |
| 锂电、机器人、消费电子 |
| 二三线有成长性的区域白酒 |

### ❌ 不适用（盈利波动大、强周期、低增长靠分红的行业）

| 行业 | 应该用什么 |
|------|-----------|
| 银行、券商 | PB |
| 强周期（有色/钢铁/煤炭） | PB（与 PE 相反） |
| 地产 | PB |
| 公共事业 | 分红估值 |

**口诀：** 有研发、有增速、增速可预测 → PEG；低增长、强周期、资产驱动 → PB。

## 九、常见误区

| 误区 | 正确做法 |
|------|----------|
| 用历史 EPS 算 PEG | G 是对未来的预期，必须对应未来一年 EPS |
| 百分数忘除以 100（写成 0.4） | 40% 就写 40，算出 PEG=100 是扯淡 |
| 只用单家券商预测 | 取五家以上券商均值或用数据库一致预期 |
| 忽视量能 | 量能是估值体系底层参数，不看量能 = 盲用 |
| 把 PEG 当万能工具 | PEG 只是工具，不判断公司好坏，不做风控 |

## 十、不同体量成长股的 PEG 用法

| 类型 | 代表 | 特征 | PEG 策略 |
|------|------|------|----------|
| **老牌龙头** | 工业富联、宁德时代、中芯国际 | 万亿级，公募主力持仓 | PEG+DCF 双模型，适合中长期，弹性有限 |
| **挑战型成长股** | 半导体龙二龙三 | 几百亿~千亿市值，弹性大 | 右侧介入，PEG<2 有价值 |
| **小市值高速成长** | 200 亿以下 | 高增速但业绩不稳 | 波段操作，增速一旦滑落第一时间止损 |
| **无盈利爆发型** | 寒武纪早期、AI/机器人/商业航天 | EPS 为负或接近零 | PEG 无意义，看烧钱放缓+营收增速+行业地位 |
| **价值崩塌型** | 主业萎缩/价格战/技术被替代 | 增速为负 | **坚决规避**，PEG 低是假象 |
