## 一、AI 芯片全景

AI 服务器中 AI 芯片是重中之重，占比最高。

### 1. 训练芯片

- 用途：训练大模型（2022 年底 ChatGPT 爆发 = 全民 AI 起点）
- 全球龙头：**英伟达**
- A 股：
  - **摩尔线程**：英伟达前大中华区总经理带队创立
  - **沐曦股份**：AMD AI 芯片前首席带队创立
- 逻辑：国内 AIDC 采购英伟达芯片时**必须配套一定比例的国产 AI 芯片** → 短期看配售逻辑，长期看国产追赶

### 2. 推理芯片

- 用途：将训练好的模型部署到实际场景（政务审批、金融风控、安防分析、工业质检、医疗影像）
- 特征：低延迟、高并发、实时响应

| A 股标的 | 方向 |
|----------|------|
| **寒武纪**（思元系列） | 视频分析、智能客服 |
| **兆易创新**（MCU + AI） | 工业控制、智能家居边缘推理 |
| **景嘉微** | 军工 GPU |

### 3. 智能终端 AI 芯片

- 用途：手机、智能家居、物联网设备的本地 AI 计算

| A 股标的 | 方向 |
|----------|------|
| **瑞芯微** | 边缘盒子核心芯片，高分辨率解码 + 算力 |
| **全志科技** | 智能车载、安防终端 |

### 4. 智能视觉芯片

- 用途：视频/图像信号处理，优化视觉算法与识别效率

| A 股标的 | 方向 |
|----------|------|
| **韦尔股份** | 指纹识别 + 人脸识别 + 虹膜识别芯片龙头 |
| **思特威** | CMOS 图像传感器，安防监控 |

### 5. 加速卡

- 用途：集成 AI 芯片 + 显存接口，插装于 AI 服务器，实现算力即插即用

| A 股标的 | 方向 |
|----------|------|
| **寒武纪** | 加速卡为主力业务（F10 营收构成可见） |
| **浪潮信息** | 与 AI 服务器深度适配，用于算力扩展 / 机房升级 / 边缘节点部署 |

## 二、AI 服务器组件拆解

### 1. PCB（主板）

- AI 服务器主板 ≠ PC 主板
- 承载多个 GPU + 超大电流 + 极高信号速率 → 工程定制款
- GPU 数量 ↑、带宽 ↑、功耗 ↑ → 层数、用材、工艺难度全面上升
- **高端 PCB = AI 服务器关键环节**

| A 股标的 | 备注 |
|----------|------|
| **盛宏科技** | 核心标的，已走出浩浩荡荡行情 |
| **深南电路** | 高速背板技术，支持 800G 信号传输 |
| **沪电股份** | 通用 AI 服务器适配 |

### 2. 高速连接器

- 连接主板、背板、高速线，保障信号低损耗传输

| A 股标的 | 特点 |
|----------|------|
| **立讯精密** | 高频高速连接器技术领先，适配液冷机房严苛环境 |
| **中航光电** | 军工级，稳定性极强 |

### 3. 背板与转接板

- 连接主板与扩展卡，实现多模块信号互通

| A 股标的 | 方向 |
|----------|------|
| **深南电路** | 高速背板技术 |
| **盛宏科技** | 更适配通用 AI 服务器 |

### 4. CPU

- 服务器的运算与控制核心，统筹 AI 任务调度
- 国内典型组合：**海光信息 + 寒武纪**（强绑定）
- 制程从 7nm 向 5nm 推进

### 5. 电源（服务器级）

- **服务器电源 ≠ AIDC 供电系统**（这是两个层面）

| A 股标的 | 特点 |
|----------|------|
| **麦格米特** | 高效电源模块市占率处于领先地位 |
| **科华数据** | 更适配 UPS + 电源一体化方案，在 AIDC 层面体现更多 |

### 6. 光模块

- 实现服务器与交换机之间的光电信号转换
- 接入层数据传输核心
- **在 AIDC 各部位都需要，整体占比远超表面数字**

| A 股「易中天」 | 特点 |
|---------------|------|
| **中际旭创** | 800G 已持续供货，1.6T（Rubin 架构）在路上 |
| **新易盛** | — |
| **天孚通信** | 光引擎封装技术领先，适配服务器板载要求 |

- 易中天均是三四千亿市值龙头，光模块在 AIDC 中的权重极重

### 7. 智能网卡

- 卸载 CPU 网络任务 → 数据分流、加密、算力调度
- 相当于**算力高速公路的收费站**

| A 股标的 | 特点 |

## 三、AIDC 算力系统

### 关键认知：所有百分比都是动态的

- GPU 功耗从 H 系列几百瓦 → Rubin 架构上千瓦 → 供电 + 散热占比持续放大
- 规模越大（如马斯克 50 万卡集群）→ 供电/散热占比越大

### 算力芯片分类

| 类型 | 说明 |
|------|------|
| GPU | 英伟达主导 |
| NPU | 神经网络专用 |
| ASIC | 专用集成电路 |
| FPGA | 可编程逻辑门阵列 |

### 国内主要超算集群分布

| 地点 | 业主 | 规模 | 特征 | 类型 |
|------|------|------|------|------|
| **内蒙古** | 字节跳动 | 30 万卡（将扩至 50 万） | 距北京 400km，低延迟低成本，液冷 | 绿电训练型 |
| **宁夏** | 亚马逊 + 腾讯 | ~20 万卡 | 绿电 100%，间接蒸发冷却 | 绿电训练型 |
| **甘肃** | 金山办公 + 华为庆阳智算 | ~15 万卡 | 算力协同 + 储能，地质稳定，电价 0.27 元/度 | 绿电训练型 |
| **贵州** | 苹果 + 华为云 | 苹果 10 万 + 华为 12 万 | 天然制冷（年均 14°C），水电丰富 | 绿电训练型 |
| **张家口** | 秦淮数据 + 阿里云张北 | ~20 万卡 | 风电直供，距京 180km，液冷，绿电 80% | 低延迟推理型 |

### 四大 AIDC 类型

| 类型 | 定位 | 典型区域 |
|------|------|----------|
| **绿电训练型** | 万卡级 AI 训练、大模型预训练 | 西北（内蒙古、宁夏、甘肃、贵州） |
| **低延迟推理型** | 服务企业 AI，极低延迟 | 北上广深 |
| **区域平衡型** | 本地训练 + 区域推理，东数西算中转 | 西安、武汉、成渝 |
| **边缘算力型** | 边缘推理、实时数据处理 | 广州、苏州、无锡、青岛 |

- 边缘算力主要适配自动驾驶和工业互联网

## 四、AIDC 供电系统

> 供电系统不是配套工程，是核心框架。占比仅次于算力系统。

### 1. 外部接入（特高压 + 变压器）

- 把城市电网电力安全稳定引入数据中心园区
- 包含：高压接入系统、主变压器、开关站、电力调度接口

| A 股标的 | 备注 |
|----------|------|
| **特变电工** | 近期因电网逻辑炒得火热 |
| **金盘科技** | 变压器，一路上涨不回头 |
| **伊戈尔** | 变压器 |

- 外部接入在供电系统中占比最高
- Rubin 架构能耗进一步上升 → 供电系统继续扩容

### 2. 直流化稳压（HVDC 高压直流供电）

- 外部电力不能直接被算力系统使用，需经直流化和稳压
- 传统数据中心：交流供电 + 多级 UPS
- AIDC 时代：功率密度上升 → **HVDC 替代方案**

| A 股标的 |
|----------|
| **中恒电气** |
| **科士达** |
| **科华数据** |

> 注意：服务器级电源看麦格米特，AIDC 级供电看科士达 / 科华数据。逻辑层次不同。

### 3. 末端供电（PDU + 母线系统）

- 把电精确送到每一台服务器、每一张加速卡
- 从简单配电功能升级为**高功率密度、高可控性技术系统**
- 占比约 5%

| A 股标的 |
|----------|
| **欧陆通** |
| **科士达**（也在布局） |

### 4. 备用电源 + UPS

- AIDC 停电不是简单的业务中断 → 模型训练失败、数据损坏、设备连锁损坏
- 安全级别最高

| 组件 | 功能 | A 股标的 |
|------|------|----------|
| **UPS** | 短时间无缝切换 + 稳压输出 | 科士达、科华数据 |
| **柴油发电机** | 中长期持续供电保障 | 科泰电源、豪泰电源 |

## 五、AIDC 散热系统

> 散热直接决定算力密度的物理上限。占比约 20%，随算力密度提升不断上移。

### 散热层级

| 层级 | 说明 | A 股标的 |
|------|------|----------|
| **芯片级液冷（冷板）** | 直接接触 GPU/加速卡发热源，通过冷板带走热量 | **英维克**、纳百川（汽车散热龙头，技术壁垒高，跨界潜力） |
| **服务器级液冷** | 设备整体热能效率 | **英维克**、**申菱环境**（两大龙头，英维克是绝对龙头） |
| **CDU（冷却分配单元）** | 控制冷却液流量/压力/温度，按需分配冷量至各机柜 | 一体式方案为主 |
| **二次冷却系统** | 冷却塔 + 冷冻水，把热量排出数据中心 | — |
| **冷却网络** | 管路、阀门、快接等基础设施 | 成本占比低但安全性不可忽视 |

### 纳百川的跨界逻辑

- 汽车散热龙头，技术壁垒高
- 可能从新能源赛道切入更高景气度的 AI 散热赛道
- 市值仅百亿 vs 英维克已近千亿 → 存在想象空间

### 核心技术趋势

- GPU 功耗：当前 700W → 未来 1000~2000W
- 传统风冷已到极限，液冷是唯一出路
- 全行业偏好**一体式散热方案**，英维克的核心壁垒在此

## 六、AIDC 网络系统

### 交换机

- 算力网络核心枢纽
- 负责服务器间、机柜间、集群间数据转发
- 速率：400G → 800G → 更高速

| A 股标的 | 备注 |
|----------|------|
| **紫光股份** | — |
| **中兴通讯** | — |

### Spine-Leaf 架构

| 层 | 说明 |
|----|------|
| **Leaf 层（叶交换机）** | 网络接入点，直连服务器和防火墙，负责本地流量交换并转发跨境流量 |
| **Spine 层（脊交换机）** | 网络高速骨干，在不同 Leaf 间进行高速路由转发 |

- 本质上就是交换机，对应紫光股份和中兴通讯

### 光纤布线系统

- 占比约 2%，但跟光模块紧密相关
- 光纤布线 = **路**，光模块 = **车**
- 集群规模扩大 → 车也要升级（800G → 1.6T）
- 「易中天」在此环节深度布局

## 七、AIDC 运维与运营

> 真正拉开差距的，不是买到更先进的 GPU，而是**谁能把算力长期、稳定、低成本地跑出来**。

- 不是资本开支，是运营支出
- 不再是「坏了再修」→ 是**提前干预风险**，保证效率
- 本质：把不确定性变成确定性，把成本曲线压成可控曲线

### 按客户类型分

| 客户类型 | 核心需求 | 运维特征 | 代表厂商 |
|----------|----------|----------|----------|
| **字节跳动系** | 超大规模集群（万卡~十万卡级） | PUE < 1.2，液冷体系稳定，全局调度、能耗优化、故障预测 | **润泽科技**、光环新网、大为科技 |
| **阿里系** | 复合功能（电商+金融+AI多场景共存） | 长期租赁、定制化建设、联合开发，弹性不如字节但抗风险能力强 | 数据港等 |
| **国资系** | 安全可靠、合规可控（政务/金融/党政军） | 国产化替代、审计合规要求极高，更像公共事业 | **铜牛信息**（军方方向） |
| **边缘算力** | 低时延推理、实时数据处理、自动驾驶/工业互联网 | 分布式部署 + 统一管理 + 轻量化运维 | **首都在线**、润建股份、网宿科技、宏景科技 |

### PUE 概念

- PUE = 数据中心总耗电量 ÷ IT 设备耗电量
- PUE = 1 → 理想极限（辅助设施零损耗）
- 绿电训练型（西北）：PUE < 1.2
- 区域平衡型：PUE ≈ 1.25

### 不同模式的盈利特征

| 模式 | 特征 |
|------|------|
| 字节系 | 高弹性、极致效率，弹性扩容 |
| 阿里系 | 稳健均衡，长期定制 |
| 国资系 | 订单周期长、现金流稳、风险低，估值靠政策和采购节奏 |
| 边缘算力 | 高周转、高弹性服务化收入。一旦某个应用爆发（视频AI/自动驾驶/工业互联网），负载率和收入弹性被迅速放大。更依赖调度能力和节点密度 |
