## DeepSeek V4 投资 Take-away

**海外小压制短期解除。** V4 预览在 coding/agent 表现较海外有差距，暂未直接冲击 Opus 4.6，杰文斯悖论前半截负面影响不如 R1。叠加 Opus 4.7 提升不明显，海外链继续受益，NV 链可能有相对超额，推荐**工业富联**。

**一切才刚开始。** 稀疏注意力、压缩 QKV、优化残差等技术创新，参考过去 MLA、DSA、postrain RL，这些引领行业的技术创新价值有望在后续 V4.1/V4.2 版本有更大体现。

**国产算力、CPU、云核心受益。** 从 V3.1 开始国产算力与国产模型相向而行，V4 进一步走向现实，昇腾等国产算力开始规模化支撑国产模型走向世界。DS 计算存储优化降低硬件门槛，agent 有望进一步普及，持续利好 AI CPU、云。继续推荐**海光信息（AI CPU+GPU）、金山云、深信服、浪潮信息**。

**国产模型不零和。** 中国 Token 走向世界是巨大市场，模型生态位会更丰富。推荐 **MiniMax、智谱**。

**非共识壁垒应用：** 杰文斯后半段最肥尾仍在应用。重视**金山办公、金蝶国际、税友股份**壁垒共识重塑与 AI 助力收入加速。

## DeepSeek V4 详细速评

4 月 24 日 DeepSeek 正式发布 V4 预览版并开源，分 Pro（1.6T/激活 49B）与 Flash（284B/激活 13B），**1M 上下文全面标配**。长上下文推理 FLOPs 降至 V3.2 的 27%，KV Cache 压缩至 10%。

**V4-Pro API 定价：** 输入 1/12 元/MTok，输出 24 元/MTok（下半年昇腾 950 上市后大幅下调）。

**三大核心技术升级：** 混合注意力（CSA+HCA）、mHC（流形约束超连接）、Muon 优化器。

**Agent 能力专项优化：** 针对 Claude Code、OpenClaw、OpenCode、CodeBuddy 等主流 Agent 产品适配优化。

**算力优化突破：** KV Cache 从 51GB 降至 5GB 以内，百万 token 实时推理成为标配。以超节点为代表的算力需求有望持续高增。

**产业链标的：**
- 算力及 Infra：海光信息、浪潮信息、金山云、深信服、工业富联、网宿科技
- AI 应用：金山办公、金蝶国际、税友股份、万兴科技、同花顺
- 模型厂商：智谱、MiniMax
