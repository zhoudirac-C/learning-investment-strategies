# 0330 午盘

> 2026年3月30日 | 低开震荡企稳回升韧性超预期 | 电力退潮 | 农业/商业航天承接 | 锂电池分歧后承接良好

---

## 盘面

- 受周末中东加剧+美股大跌影响全线低开→下探震荡后企稳回升→冲高回落再震荡走高
- 韧性十足，强度略超「全天水下震荡」预期。A 股连续在外围大跌冲击下低开走强→**抗跌属性凸显**
- 3870 有支撑，不要在支撑位割肉（要减在压力位如上周五）

## 情绪

- 短线情绪偏弱，核心=电力板块集体退潮拖累。高位滞涨后补跌，退出主线影响力
- 此次退潮早于预期（未出现共识后常见的补涨/大市值鱼尾加速行情）→负反馈后续大概率扩散

## 板块

| 方向 | 表现 |
|------|------|
| **农业/粮食** | 地缘冲突相关方向领涨 |
| **商业航天** | 电力流出资金主要流向→延续上周四表现。但进场资金多为浮筹→后续大概率承压。4 月密集发射窗口+深度调整后刚企稳→**重要轮动板块** |
| **光纤光缆** | 再度领涨带动光通信。后续有望向保偏光纤/空心光纤等高端化演绎补涨。法拉第旋片/磷化铟衬底预期差 |
| **医药** | 美诺华加速带动延续强势，超跌反弹强度可观 |
| **锂电池** | 昨日新启动后早盘随大盘调整兑现，但**承接力度良好**，核心公司无明显亏钱效应。刚形成板块阵型的新方向→分歧后仍有反包机会 |
| **电解铝** | 中东铝厂遭袭（全球约一成供应）+欧洲/非洲高电价产区持续减产（潜在 150~200 万吨/年）。华通线缆安哥拉项目吨铝利润有望超 1 万元 |

## 策略

- 当前无明确强主线，板块轮动为主。重点关注**锂电池、商业航天、AI 硬科技**三大方向轮动机会
- 不追高，等大盘或板块分歧时参与更具安全性
