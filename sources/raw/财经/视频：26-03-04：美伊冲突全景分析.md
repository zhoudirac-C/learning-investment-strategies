## 三、战场实况：非对称消耗战

**成本严重不对等：**
- 伊朗无人机：2-5 万美元/架
- 美军拦截导弹：400+万美元/枚
- 视频记录：伊朗 1 枚弹道导弹 → 美军 11 枚拦截全部脱靶

**美国军工体系致命缺点：** 冷战后的精密武器体系适合秀肌肉，不适合打消耗战。拦截弹严重短缺 → 需从亚洲调配库存 → 直接削弱亚太战略部署 → 给中国创造战略窗口期。

**伊朗反击范围：** 以色列多地遭导弹无人机袭击；迪拜美国领事馆被击中；卡塔尔机场、美军基地、沙特利雅得 CIA 站点、科威特、约旦全部遭袭。黎巴嫩真主党协同发射。

## 四、霍尔木兹海峡：全球经济的命门

| 数据 | 数值 |
|------|------|
| 全长 | 161km，最窄处 34km，双向通道仅 **6.2km** |
| 日均运输量 | **1670 万桶**（全球 1/4 海上石油贸易） |
| LNG 贸易 | 全球 1/5 |
| 铝出口 | 全球 15% |
| 铁矿石 | 全球 15% |
| 化肥 | 全球 1/3 |

**沙特/伊拉克/科威特/阿联酋/伊朗的石油绝大部分走这里。卡塔尔全球最大 LNG 出口设施完全依赖此航线。**

### 替代方案根本不够

| 替代 | 运力 | 缺口 |
|------|------|------|
| 沙特东西管道 | 500 万桶/日 | — |
| 阿联酋管道 | 150 万桶/日 | — |
| 合计 | 650 万桶/日 | vs 海峡 1670 万桶 = 只能补 **1/4** |

> **断航超两周 = 不再是金融风险，是全球经济的巨大风险。粮食、能源、通胀全面失控。所有缓冲机制都扛不住。**

## 六、金融市场反应

**典型地缘冲突模板：油和美元一起涨，其余资产一起跌。**

| 资产 | 表现 |
|------|------|
| 道指 | 一度跌 900 点，收 -0.83% |
| 标普 500 | 收 6816，-0.94% |
| 纳指 | 最深 -1.8%，收 -1.02% |
| 原油 | 单日 +4%，累计自开战 +10%，收 80 上方 |
| 黄金 | 日内暴跌 300 点，勉强守 5000（卖金补仓） |
| 美元 | 连续第二日上涨，站上 99 |
| 美债收益率 | 盘中突破 4.1% 后回落（通胀担忧 vs 避险买入的矛盾拉扯） |
| 降息预期 | 59bp→46bp，首次降息从 7 月推迟到 9 月 |

**标普 6800 是结构分界线：** 上方 = 回调；有效跌破 = 结构性下行。若速战速决预期破灭 → 标普合理估值 18-19x → **15-20% 下跌空间。**

## 七、中国的处境与选择

### 四大风险

| 风险 | 内容 |
|------|------|
| **能源安全** | 12% 原油进口直接来自伊朗，更大比例从沙特/伊拉克走海峡；油价暴涨→输入性通胀→与国内通缩螺旋对冲→压缩货币宽松空间 |
| **地缘信誉** | 伊朗是上合+金砖核心伙伴，美国公然颠覆但我们无法提供实质安全支持→动摇全球南方国家对中国的安全承诺信心 |
| **平衡外交两难** | 沙特对华原油出口量已超伊朗；帮伊朗=损害海湾国家关系；不帮=失去反美阵营核心 |
| **大宗商品冲击** | 铝/铁矿石/化肥/食用糖全线短缺涨价 |

### 四大机遇

| 机遇 | 内容 |
|------|------|
| **亚太窗口期** | 美国从亚太（台海）抽调弹药和资源去中东；最新战术体系暴露=情报红利 |
| **话语权提升** | 美国=丛林法则+全球不稳定根源；中国=国际法维护者+和平推动者 |
| **战后重建红利** | 中国是全球唯一具备全产业链基建能力+充足资金+大规模投资意愿的国家 |
| **去美元化加速** | 俄罗斯获油价财政收益，中俄合作更稳固；人民币国际化窗口 |

### 克制的原因

- 伊朗 90% 石油出口依赖中国市场，但只占中国进口 12% → 关系严重不平衡
- 坚持不干涉内政原则，吸取美国中东泥潭教训
- 美国制裁硬约束：和伊朗大规模合作 = 面临次级裁决
- 已提前布局：战略石油储备 + 电动汽车/可再生能源降低石油消费依赖
