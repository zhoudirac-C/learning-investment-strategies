# 博主方法论总纲

## 来源范围

本页来自 `sources/raw/财经` 391 篇文档的全量梳理，重点沉淀多次出现、可复用、可检验的方法论。单日判断不直接升级为长期规则，只有跨阶段重复出现并能解释不同市场环境的内容进入本页。

## 怎么学习这套体系

如果你是小白，不建议直接从 raw 或 claims 开始看。raw 是原始材料，信息密度高但顺序很散；claims 是观点证据卡，适合核对来源，不适合入门。更好的顺序是先建立骨架，再用每日复盘和原文去验证。

建议阅读顺序：

1. 先读本页，理解博主体系的主干：周期、主线、个股、仓位、纪律。
2. 再读方法论专题：[交易纪律](./交易纪律.md)、[仓位管理](./仓位管理.md)、[主线判断](./主线判断.md)、[分歧中找机会](./分歧中找机会.md)。
3. 然后读市场判断专题：[修复性质判断](../市场分析/修复性质判断.md)、[指数调整](../市场分析/指数调整.md)、[量能判断](../市场分析/量能判断.md)、[科创50虹吸](../市场分析/科创50虹吸.md)。
4. 再读主线产业专题：[半导体产业链](../市场分析/半导体产业链.md)、[长鑫存储产业链](../市场分析/长鑫存储产业链.md)、[PCB与先进封装](../市场分析/PCB与先进封装.md)。
5. 最后读每日案例，把规则放回盘面里验证，例如 [2026-05-20复盘](../每日复盘/2026-05-20.md)、[2026-05-21复盘](../每日复盘/2026-05-21.md)、[2026-05-22早盘](../每日复盘/2026-05-22.md)。

每天看盘时，用五个问题把信息串起来：

1. 现在是什么周期：主升、分歧、修复、退潮，还是高位宽幅震荡。
2. 今天的修复是真修复还是弱反抽：谁在带，量能是否配合，核心票是否主动。
3. 主线是谁：是半导体、AI硬件、存储、PCB/先进封装，还是低位杂题材乱动。
4. 自己的票是什么身份：主线核心、中军、弹性、补涨、后排，还是破趋势弱势股。
5. 操作怎么落地：持股、做T、低吸、减仓，分别对应什么确认信号和证伪信号。

## 总原则：先周期，再主线，再个股

博主的分析顺序不是从个股开始，而是从市场状态开始：

1. 判断市场周期：主升、混沌轮动、强分歧、退潮、冰点、修复。
2. 判断指数和量能：关键位、成交额、顶部结构、底部结构、承接质量。
3. 判断主线：是否有产业逻辑、是否有资金主动扩散、是否由核心带队。
4. 判断个股身份：核心、容量、趋势、补涨、后排、案例或过期。
5. 最后给出操作：持有、低吸、减仓、观望、纠错买回或放弃。

这套框架的关键是防止“只看个股分时”或“只看单日涨跌”。市场状态错误时，个股再好也容易被阶段风险吞掉。

## 周期框架

### 主升

主升阶段的典型特征：

- 指数趋势向上，成交额持续放大或保持高位。
- 主线板块持续扩散，核心票和容量票共同走强。
- 赚钱效应从前排扩散到补涨，但尚未出现大范围亏钱效应。
- 技术顶部信号未形成，或形成后被量能和主线强度纠错。

策略：持股为主，围绕主线核心做回调，不轻易被盘中波动洗出。

### 混沌轮动

混沌轮动阶段的典型特征：

- 热点多但持续性弱。
- 同一方向难以连续两天承接资金。
- 指数可能不弱，但板块轮动快，可预见性低。
- 资金在科技、资源、防御、低位题材之间快速切换。

策略：买阴卖阳，低吸核心，反抽见好就收，不把短线脉冲当趋势。

### 强分歧

强分歧阶段的典型特征：

- 高位放量杀跌，获利盘集中兑现。
- 主线第一次出现明显亏钱效应。
- 防御方向可能护盘，但主线是否结束仍需承接确认。
- 次日重点不是看反弹，而是看关键位、量能、跌停家数和主线回流。

策略：不恐慌，不追高，只在承接成立后围绕主线核心低吸。

### 退潮

退潮阶段的典型特征：

- 高位股连续负反馈，跌停家数增加。
- 前期主线补涨股和后排先补跌。
- 活口容易只是退潮活口，逆势并不等于新周期核心。
- 资金偏向防御、业绩、涨价或低位避险。

策略：降低仓位，远离杂毛和后排，等待冰点或新主线确认。

### 冰点与修复

冰点不是“跌了很多”的同义词，而是情绪、筹码和仓位结构达到阶段性极端。常见观察包括跌停家数、连板高度、晋级率、成交额、恐慌释放和主线核心承接。

修复成立需要：

- 指数关键位不再继续破坏。
- 成交额从恐慌放量转向降温或强承接。
- 跌停家数明显下降。
- 主线核心主动修复，而不是只靠银行、白酒、消费护盘。

### 拐点确认的节奏判断（假摔判定）

拐点的确认**不看单日，看连续两日的走势组合**：

| 路径 | 第一日 | 第二日 | 结论 |
|------|--------|--------|------|
| 路径一：假摔确认 | 不强修复（震荡或下跌） | 强修复 | 拐点延后确认，方向不变 |
| 路径二：直接修复 | 强修复 | 趋势延续 | 拐点当日确认 |

**核心原则**：
- 两种路径都指向"拐点确认"，差异只在时间节奏
- 若第一日不强修复且第二日也不修复 → 假摔判定证伪，需重新评估
- 拐点节点的操作核心是**判断明天方向**，不是押注今天涨跌

## 量能框架

量能在博主体系中承担三种功能：

1. 判断风险偏好：成交额越高，市场对成长和高估值方向的容忍度越高。
2. 判断承接质量：放量上涨和缩量反弹含义不同，放量下杀破坏力更强。
3. 调整估值标准：PEG、PE、PS 等估值工具不能脱离市场成交额和流动性环境。

常见用法：

- 主升中量能温和放大，是趋势延续信号。
- 高位放量但指数不能收复关键位，是筹码继续交换信号。
- 极度缩量叠加窄幅震荡，往往意味着变盘临近。
- 量能不支持大市值持续强势时，容易进入缩容炒作。

## 技术结构框架

博主经常使用日线高 9、高 10、30/60/90/120 分钟顶部结构、底部结构、DIF 和布林线等工具。其用法不是机械买卖，而是服务于仓位和纠错。

规则：

- 顶部钝化和高 9 触发风险意识，降低追高性价比。
- 顶部结构消失时，需要纠错买回，而不是固守原减仓判断。
- 底部结构形成时，先小仓位试错，再看量能和主线共振提高仓位。
- 技术信号必须与情绪周期、主线强度和成交额共同验证。

### 波浪理论的应用规则

博主在特定市场阶段使用波浪理论进行定量判断，遵循以下原则：

1. **时间空间锁定一个确定项**：波浪理论中时间和空间往往只能确定其一。B浪空间可确定（如4200点目标），时间不确定；C浪时间可确定（如6月中下旬结束），空间不确定。不必强求两者同时精确。
2. **拐点判断的概率思维**：不直接断言"当日就是拐点"，而是结合W底形态、外围消息（如英特尔盘前走势）给出"成功率更高"的判断，承认市场的不确定性。
3. **形态验证优先**：单个W底成功率低，第二个类似形态成功率显著更高。形态需与量能、主线、外围消息共同验证。
4. **ABC调整后的布局窗口**：C浪调整结束后往往出现"黄金坑"，是中线布局主线的最佳时机。

## 主线框架

主线不是涨幅最大，而是同时满足产业逻辑、资金强度和持续扩散。

有效主线常见特征：

- 有明确产业催化或供需变化。
- 前排核心和容量标同时走强。
- 板块内部从核心向中游、上游、材料、设备、应用有序扩散。
- 分歧后能承接，而不是一天高潮后次日直接退潮。

主线后期的风险信号：

- 核心不再主动领涨，后排补涨开始加速。
- 全链条补涨闭环完成，低位杂毛轮番冲高。
- 技术顶部钝化和情绪高潮同步出现。
- 资金开始在防御方向和低位题材中找避险。

## 个股身份框架

| 身份 | 特征 | 处理 |
| --- | --- | --- |
| 核心龙头 | 产业逻辑最硬、资金辨识度最高、能带板块 | 分歧低吸或持股跟踪 |
| 容量标 | 成交额大，机构和趋势资金可参与 | 用于判断主线强度 |
| 趋势票 | 基本面或产业逻辑持续兑现，走势稳 | 拉长持股周期 |
| 补涨票 | 主线扩散后低位跟随 | 只做节奏，不恋战 |
| 后排跟风 | 无独立逻辑，靠情绪推动 | 高潮期风险最大 |
| 退潮活口 | 分歧初期暂时抗跌 | 不追，观察是否补跌 |
| 复苏活口 | 充分分歧后率先修复并带动扩散 | 重点跟踪 |
| 案例票 | 用于解释逻辑，不构成持续跟踪 | 记录但不自动纳入股票池 |

## 仓位框架

- 主升趋势未破坏：以持股为主，围绕核心处理。
- 主升后期出现高 9、顶部钝化、全链条补涨：收缩仓位，保利润。
- 混沌轮动：轻仓打野，买阴卖阳。
- 退潮：控制仓位，等待冰点。
- 底部结构和主线共振：从小仓位试错逐步提高。
- 误判后信号被纠错：按规则买回，不用情绪对抗盘面。

## 输出要求

后续使用本方法论分析任何股票时，必须输出：

- 市场状态。
- 板块阶段和主线身份。
- 个股身份。
- 支撑结论的 source path、claim id 或案例。
- 操作条件和证伪条件。
