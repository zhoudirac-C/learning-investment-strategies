# AI上游涨价链

## 核心传导链条

```
AI需求爆发 → 产能优先向高端倾斜 → 高壁垒导致供给刚性 → 全品类涨价轮动
```

## 2026-05-23：Rubin BOM价值扩散

5月23日视频进一步把 AI 上游涨价链从“材料涨价”扩展到“下一代 AI 机柜 BOM 价值扩散”。核心依据是摩根士丹利对 Vera Rubin / VR200 NVL72 机柜的拆解：AI服务器价值量不再只由 GPU 决定，而是向存储、PCB、MLCC、ABF基板、电源、液冷、ODM组装测试等组件扩散。

关键变化：

| 环节 | 变化 | 交易含义 |
|------|------|----------|
| 存储 | VR200内存价值占比显著提升 | 中期逻辑强化，但前期涨幅较大，需要横盘消化后再观察 |
| PCB | 受层数提升、CCL升级和新增模块驱动，价值量增幅突出 | 高端PCB、HDI、CCL、高速材料继续纳入AI硬件主线 |
| MLCC | 绝对金额不大但增速快 | 高容高可靠产品供给紧，可能成为AI服务器通胀型材料 |
| ABF基板 | 单颗芯片价值提升，互联芯片数量增加 | 先进封装底座，跟随Rubin/Rubin Ultra继续抬升价值量 |
| ABF材料/增层膜 | ABF基板上游关键材料，国产替代难度高 | 重点看类ABF增层膜、客户验证、小批量和营收确认 |
| 电源/液冷 | 功率密度提升，液冷从选配变标配 | HVDC、连接器、铜排、散热、电力电阻等中长期受益 |
| ODM | 机柜复杂度提升，组装测试绝对价值增加 | 毛利率未必升，但利润池和现金流模式可能改善 |

这条线的学习重点是：**从GPU单点涨价，转向拆BOM找价值量增幅最快、位置相对低、供给壁垒高的环节**。

## 2026-05-24/25：Rubin扩散进入材料、工艺与耗材层

5月24日动态和5月25日AI PCB研报把 Rubin BOM 扩散进一步细化：PCB 已经不只是成品板涨价，而是高速互联架构升级带来的材料、工艺、设备和耗材系统性重估。

核心链条：

```
SerDes/NVLink速率升级 -> 损耗预算收缩 -> 层数增加 + 材料升级 -> 工艺/设备/耗材壁垒上升
```

新增观察层：

| 层级 | 方向 | 逻辑 |
|------|------|------|
| 高多层板 | Midplane、Switch板、HPM主板 | Rubin 从线缆转向 PCB Midplane，Rubin Ultra 进一步到整柜级高层数背板 |
| 高速材料 | M8/M9 CCL、碳氢树脂、PPO、Q-Glass | 224G/448G 高速信号要求低介电、低损耗和高可靠性 |
| 铜箔 | HVLP4、HVLP5 | 表面粗糙度决定高频信号损耗，是高速板硬约束 |
| 工艺设备 | mSAP、LD曝光、图形电镀、UV激光钻孔 | 高密度线路和高阶HDI良率取决于工艺能力 |
| 耗材 | PCB微钻、铣刀 | 板更厚、材料更硬、孔更多，钻针消耗和单孔成本提升 |
| 光互联延伸 | CPO、DCI光纤 | PCB分歧后可能承接资金，属于柜外互联预期差 |

学习处理上，越靠上游和细分，越不能只看“概念映射”。验证层级从低到高为：

1. 样品/送样；
2. 客户认证；
3. 小批量供货；
4. 收入确认；
5. 持续订单和利润兑现。

## 2026-05-24：鱼尾与扩散阶段风险

周复盘把 PCB、MLCC 定义为周末已充分发酵的方向。产业逻辑仍强，但周一若一致高开，被顶到高位后普通投资者追高性价比下降。

这补充了 AI 上游涨价链的交易纪律：当资金不断挖低位涨价、低位材料和小票补涨时，说明主线从核心进入中后段扩散。机会仍在，但要优先等分歧，看承接，而不是把研报发酵后的高开当成买点。

## 长周期风险：BOM涨价也是ROI压力

5月23日另一条长视频从“2028 AI资本狂潮终局”角度补充了风险视角：VR200机柜涨价既是AI硬件价值量扩散的景气信号，也是摩尔定律衰减后客户ROI压力上升的风险信号。

这意味着AI上游涨价链的交易不能只看“涨价”和“价值量提升”，还要同步检查：

- 云厂商资本开支是否继续上修；
- AI公司收入和Token需求是否能覆盖算力成本；
- GPU二手价格是否稳定；
- HBM供需是否仍紧；
- GPU租赁公司融资利差和利用率是否恶化。

如果上述长期指标没有共振恶化，涨价链仍按产业扩散和订单兑现跟踪；如果多项同步恶化，就要下调AI硬件链估值容忍度。

## 2026-05-25：低位涨价方向启动后的尾声风险

5月25日动态复盘确认，资金继续沿“AI需求爆发 -> 产能向高端倾斜 -> 行业供给收缩 -> 全品类涨价”的链条向低位方向扩散，电力、功率器件、变压器等开始批量异动。

这说明第三波低位涨价方向开始被资金挖掘，但交易纪律也同步变化：

- 有先手可看承接和扩散；
- 没有先手不追集体加速；
- 若低位补涨大面积启动，说明扩散进入后段，要防补涨尾声兑现；
- 低位补涨强不等于主线切换，仍要看AI科技核心是否保持承接。

## 核心标的

### 风华高科（MLCC）
- AI服务器MLCC用量较传统服务器激增5-10倍
- 月产能约350亿颗，祥和工业园高端基地持续释放产能
- 2026年内高端产品占比计划提升至50%
- 已批量导入国内头部AI服务器及机器人客户

### MLCC三类跟踪

| 类别 | 代表 | 观察点 |
|------|------|--------|
| 成品龙头 | 三环集团、风华高科 | 海外龙头涨价后，中高端或中低端需求是否外溢给国内厂商 |
| 上游设备/材料/零部件 | 昀冢科技、洁美科技、国瓷材料、达利凯普、博迁新材 | MLCC扩产、离型膜、载带、电子陶瓷业务是否带来业绩弹性 |
| 弹性公司 | 利和兴等 | 高端MLCC供应链参与度、订单验证和业绩确定性 |

> 注意：昀冢科技是按MLCC产业链上下文确认的精密电子零部件方向，不是云从科技-UW。

### ABF/FC-BGA载板跟踪

| 类别 | 代表 | 观察点 |
|------|------|--------|
| 容量核心 | 深南电路 | ABF/FC-BGA封装基板、高端PCB、服务器高速背板业务的订单和产能释放 |
| 弹性布局 | 兴森科技 | BT/ABF载板产能、良率、客户验证和业绩拐点 |
| 高端PCB | 沪电股份、生益电子、胜宏科技等 | AI服务器高阶PCB、HDI、交换板和背板需求 |
| 上游材料 | 生益科技、联瑞新材等 | 高速CCL、球形二氧化硅、先进封装材料涨价和国产替代 |

### ABF材料/类ABF增层膜跟踪

| 类别 | 代表 | 观察点 |
|------|------|--------|
| 类ABF增层膜 | 天和防务/天和嘉膜、莲花控股/纽菲斯新材 | FCBGA载板验证、客户认证、小批量供货、收入确认 |
| ABF/BT基材协同 | 中京电子/盈骅新材 | ABF膜/BT基材能力、送样验证、参股比例和与IC载板业务协同 |
| 替代路线 | 华正新材 | CBF积层绝缘膜的终端验证、系列化、量产进度和营收占比 |
| 合作研发 | 宏昌电子/晶化科技 | 增层膜新材料研发、特定产品销售和批量化落地 |

这层属于“国产替代验证池”，交易上不能按成熟 ABF 材料量产来直接定价。优先级从高到低看：客户认证 > 小批量供货 > 收入确认 > 样品/合作公告 > 概念映射。

### 海星股份（电极箔）
- 全球前二电极箔企业
- AI服务器供电架构向800VDC升级驱动需求井喷
- 市场规模：2026年25.9亿→2028年86.6亿→2029年突破100亿
- 2025年底总产能约5400万平方米，2026Q3全面达产将超6000万平方米

## 2026-05-26：Rubin链不证伪但等待分歧

5月26日早盘补充将PCB、MLCC、超级电容等Rubin链定义为“资金介入较深、短期不会快速证伪”的方向，但交易纪律从周末发酵后的进攻，切换为分歧后的低吸。

这条更新的含义：

- Rubin链仍是AI硬件价值扩散主线，不能因单日后排回落就判定结束；
- 不追一致高开和情绪后排，重点等核心容量票分歧后是否承接；
- 若韬定律国产链强势，Rubin链可能先让位；若国产链分歧，Rubin核心能否回流是AI硬件韧性的观察点。

## 逻辑优势

- 位置相对CPO不拥挤
- 逻辑嵌入AI硬件大主线
- 后续高低切时可能反复轮动
