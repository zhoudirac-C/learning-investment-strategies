## 一、核心判断：AI 基础设施是不可逆的资本循环

4 月 12 号彭博社文章《即使市场剧烈震荡，人工智能巨头仍在继续前行》揭示了一个关键拐点：摩根士丹利维持今年发行 **4000 亿美元高等级债券**的预期，用于支持超大规模数据中心和 AI 相关投资。

**商业模式闭环已形成：** 大厂发债 → 建数据中心 + 买 GPU → 出租算力 → 还债。信贷投资者愿意买这些公司的债，AI 从「信仰投资」变成了可以算清楚的一笔生意。

**自我强化的资本循环：** 越有资金为 AI 基础设施买单 → 产业越有能力扩张 → 扩张越快 → 市场越相信主线能持续。在中东局势反复、油价上行、通胀预期抬头的背景下，华尔街依然为 AI 产业链持续融资，说明在全球资本眼中，AI 基础设施是必须参与的「B 选项」。

## 二、AIDC：2026年建设拐点

### 2.1 AIDC vs 传统 IDC

| 维度 | 传统 IDC | AIDC（智算中心） |
|------|---------|-----------------|
| 定位 | 普通机房 | 专为 AI 服务的新型数据中心 |
| 单机柜功率 | 4kW | **30kW+** |
| 散热 | 风冷/普通空调 | **液冷刚需** |
| 设计要求 | 常规 | 供电/散热/网络/机房整体重做 |

行业已从小规模部署进入 **GW 级别园区**时代（大同、芜湖、内蒙超大规模智算中心）。

### 2.2 拐点的两大驱动力

**驱动力一：国产算力放量**

DeepSeek V4 高度适配国产算力卡，国产算力出货量从试点规模进入**几百万张级别**。之前因没有卡被卡住的 AIDC 项目可大规模推进。每一张国产算力卡背后对应服务器、电源、散热、液冷、交换机、光模块、存储、机柜、配电运营的全套采购。

**驱动力二：H200 进口有望放开**

5 月中旬中美谈判，重点之一是放缓对英伟达芯片的约束。若谈成，H200 + 国产算力卡搭配方案落地，AIDC 建设速度将明显加快。

### 2.3 三大直接结果

1. 招投标量明显放大，真实订单出现
2. GW 级园区常态化（未来值钱的是整片园区的承载/电力/配套能力）
3. 高功率机柜占比持续提升，单个项目价值量因规格升级而提高

## 三、全球算力结构变化：从训练中心到推理中心

过去追求模型能力（训练越大越强），现在转向推理吞吐能力。训练是阶段性的，推理是持续性的。随着 Agent/多模态/工具调用普及，单次任务的计算量远超单轮文本回答（思维链模式单次算力消耗增加 **150 倍**）。

**中国算力产业三阶段路径：**

| 阶段 | 内容 | 代表方向 |
|------|------|----------|
| 第一步 | 云厂商/算力租赁平台承接推理需求（谁有可用算力谁先变现） | 宏景科技、利通电子 |
| 第二步 | 国产芯片 + 企业私有化部署 | 寒武纪、华为海思、摩尔线程、沐曦股份 |
| 第三步 | 边缘端算力 | 网宿科技（长期锁仓方向） |

## 四、算力通胀：供需错配驱动的涨价周期

**涨价证据链：**
- B300 租赁价格：3 月涨 35%，4 月仍在涨
- 优刻得、网宿、首都等头部厂商全线涨价 20%-40%
- 阿里云 Coding Plan 已开始限购
- 腾讯云开始涨价

**中国独有的成本优势：** 绿电区域（风电/光伏）的 IDC 建设 → 电价低 → token 成本低 → 价格更低 → 用户更多 → 调用量放大 → 规模效应 → 单位成本进一步摊薄。**成本优势→份额优势→成本优势的飞轮**，是中国模型通过 API 把服务卖到海外的核心竞争力。

## 五、算力租赁：从重资产收租到平台型生意

### 5.1 三层壁垒（马太效应）

| 壁垒 | 说明 | 案例 |
|------|------|------|
| 系统交付能力 | 万卡集群级别的整体方案，不是小规模采购 | 头部厂商优势 |
| 拿卡能力与交付速度 | 稳定渠道获取高端卡（有合规风险） | 协创数据曾因供应商被抓爆雷 |
| 资金实力 | 重资产行业，订单越大垫资越多 | 宏景科技 26 年新增授信 500 亿，协创数据 1000 亿 |

结论：订单向少数头部玩家集中，蹭概念的公司有暴雷风险。

### 5.2 商业模式升级

**传统模式：** 买设备建机柜 → 按月出租 → 慢慢回本（天花板有限，毛利有限）

**未来模式：** 固定收租 → **参与 token 分成 + 模型服务收费**

算力太稀缺，掌握算力资源的公司拥有了定价权。大厂为保障模型不崩溃必须锁定算力，每卖出一份套餐 token，一部分分给算力租赁公司。**美国 CoreWeave 已和算力租赁厂商谈成分成，国内头部大模型公司正在推进。**

### 5.3 财务重估：设备残值的重新认识

传统会计折旧下，几年后账面残值很低。但在算力供需紧缺环境下，服役几年的高端卡依然可出租。**账面残值（15%）vs 真实经济价值（机构预测接近 70%）** 存在巨大偏差。一旦银行和融资机构按更高价值看待这些资产 → 融资成本下降 → 净利率中枢上移。

### 5.4 利润释放的三层逻辑

| 层次 | 驱动因素 | 特点 |
|------|----------|------|
| 第一层 | 租金上涨（供需缺口） | 最直观 |
| 第二层 | 财务费用下降（残值重估 → 融资条件改善） | 影响更深，优化净利率结构 |
| 第三层 | 商业模式升级（租金 → token 分成） | 估值逻辑重算，净利率可能走向 20%+ |

## 六、存储：被低估的 AI 基础设施环节

- 模型越大 → 内存带宽和容量要求越高 → HBM 和 DDR5 同步升级
- 多模态/Agent/企业知识库 → 数据类型暴增 → 存储密度/读取速度/稳定性要求大幅提升
- AIDC 建设离不开高速存储和并行文件系统，否则算力被数据进出瓶颈浪费
- 关注重点不是手机/PC 终端恢复，而是**服务器 DRAM 和数据中心 NAND**
- 存储中长线确定性极强，是 AI 基础设施不可缺的一环

## 七、AI 超节点架构与交换芯片

**传统架构问题：** 八卡一组放在同一机柜，不同服务器间通信耗时，80% 时间在等数据，真正计算时间不到 30%。

**AI 超节点：** 将数百上千张卡打通为一个超级整体（Scale-up 模式），通过交换芯片实现：
- 每秒处理几十亿条数据传递
- 精准直达最短路径
- 100% 数据准确送达
- 一份数据一次同步给所有计算单元

**架构切换：** Scale-out（传统 IDC，分散连接）→ **Scale-up**（AI 集群，紧密协同）。切换后交换芯片需求跨越式放大。

**国产格局：** 全球龙头博通（英伟达自研自用不对外供应），国内对标盛科通信、华为海思、中兴微电子为第一梯队，芯原股份等也在追赶。

## 八、光通信产业链全景

| 方向 | 逻辑 | 当前状态 |
|------|------|----------|
| **OCS 光路交换** | Google TPU V7 催化，高规格 AI 集群对光互联要求升级 | 最受关注 |
| **800G 光模块** | 集群规格越高，模块等级要求越高 | 持续高景气 |
| **高端 PCB** | 承载高规格通信和高速数据链接的核心底板 | 工艺要求持续提升 |
| **液冷** | 高功率芯片标配，前期回调充分 | 回调到位即是介入机会 |
| **光芯片/光纤光缆** | 需求配套，但相对偏弱 | 短期走弱，等调整 |
| **存储芯片/电力设施/燃气轮机** | 北美出口电力设备链条 | 重点关注 |

```
AI应用持续爆发（token快速消耗）
  → 大厂资本开支砸向AIDC（GW级园区常态化）

          → 反过来继续强化国内推理算力需求
            → 正向循环
```
