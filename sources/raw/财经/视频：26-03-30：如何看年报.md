# 如何看年报

> 2026年3月30日 | F10实战 | 以金风科技为例

---

## 一、年报阅读顺序

### 第一步：管理层是否出问题

看没有出席的董事。如有独立董事因「工作原因」缺席，需核实。关键岗位董事无故缺席 → 可能爆雷（有先例）。

### 第二步：主营业务与产品

看营收、净利润、行业地位。
- 案例公司：年营收 730 亿（+28.79%），净利润 27 亿（+19.1%）
- 国内市场份额 21%，全球 17%，国内连续 15 年第一，全球连续 4 年第一
- **当行业龙头业绩出现大幅增长时，意味着整个行业可能迎来转机**

### 第三步：F10 财务数据

**毛利/净利：** 同行业对比。制造业不能拿高科技或茅台的标准去比。

**偿债能力：** 流动比 + 速动比，与行业平均对比。案例公司因 to B 业务（下游央企/国企），应收账款 300+亿，流动比和速动比低于行业均值，但结算模式（预付款→进度款→交付款）决定了这并非异常。只要没有重大变动就 OK。

**运营能力：** 应收账款周转率 + 存货周转率。健康即可。

**杜邦分析（ROE）：** 结合周转率，整体健康即可。

### 第四步：产品结构变化

案例公司实际出货 26000MW（+65%）。结构变化：6MW 以下 -50%，6-10MW 及 10MW+ 大幅增长 → 产品向大型化高端化升级，毛利净利改善。

## 二、全产业链联动

**龙头业绩好转 → 全行业变化。**

以风电为例：
| 环节 | 代表 | 表现 |
|------|------|------|
| 整机（中游） | 金风科技 | 营收 730 亿，净利 27 亿 |
| 叶片（上游） | 中材科技 | 业绩增长比金风更亮眼 |
| 塔筒（上游） | 大金重工 | 业绩极好，走趋势大牛 |

> **研究产业的逻辑：先找行业龙头 → 沿龙头产品线向上游挖原材料 → 全产业链贯通。**

## 四、新业务深度拆解：绿色甲醇

### 背景

- 公司 2025 年签订 15 万吨，在手订单 75 万吨；兴安盟项目（50 万吨，一期 25 万吨）2025 年 9 月首次开车成功
- 与马士基签订年产 50 万吨长期采购协议（支持首批 12 艘甲醇双动力船）
- 中远海运预测：2030 年航运业绿色甲醇需求达 **1 亿吨**

### 成本分析

**两种技术路径：**
| 路径 | 碳排放降低 | 代表 |
|------|-----------|------|
| 生物质耦合制甲醇 | **95%+** | 金风科技（风电+生物秸秆） |
| 纯电制甲醇 | 70-90% | 光伏制氢路径 |

**金风科技的成本优势：**
- 选址兴安盟：一类风能资源区 + 东北玉米秸秆充裕 + 靠近营口/葫芦岛港口
- 风电 24 小时连续供电 vs 光伏仅白天且季节性强 → 风电耦合制成本更低
- 风电制甲醇成本约 2200 元/吨（vs 光伏制甲醇 ~3800 元/吨）

### 利润测算

| 参数 | 数值 |
|------|------|
| 售价 | ~6000 元/吨（当前 7200+） |
| 风电制总成本 | ~4500-5000 元/吨 |
| 单吨利润 | ~2600 元 |
| 一期 25 万吨年利润 | **~6.5 亿** |
| 二期 50 万吨达产年利润 | **~13 亿** |

**主营业务净利 27 亿 + 绿色甲醇 6.5-13 亿 = 可观的利润增量。**

### 竞争格局

- 全球最大生物质耦合绿氢项目 = 兴安盟（唯一）
- 马士基在摩洛哥的大项目 2028-2029 年才投产
- **金风科技可吃三年独占期**
- 且生物质路径碳排放最低 → 产品最高端 → 理应卖更高价

## 五、投资哲学：等大机会

**看年报的目的：** 找到业绩好、有增量、不受突发事件影响的龙头公司。

| 现在 | 未来 |
|------|------|
| 每天 T 一 T、吃小肉 | 赚产业趋势 + 戴维斯双击的大钱 |
| 环境不好，小仓位做反弹 | 风险出清后重仓 all in |

**如果四月真有一波杀跌，把金风科技从 1000 亿干到 600-700 亿：**
```
1000亿 → 2000亿 = 1倍
600亿 → 2000亿 = 3-4倍
```

> 「我希望你能把我干下来，你干下来我后面利润更大。」

## 六、年报实战总结

| 步骤 | 内容 |
|------|------|
| 合规安全 | 董事出席情况、是否频繁换会计师事务所 |
| 管理层 | 是否靠谱 |
| 主营业务 | 靠什么赚钱、行业地位、核心竞争力 |
| F10 三张表 | 利润表 + 资产负债表 + 现金流量表 |
| 行业估值 | 不同行业用不同估值方法（PE/PEG/PS/PB） |
| 新业务 | 对业绩增量多大、赛道天花板、竞争格局 |
| 持续跟踪 | 季报更新后一步步佐证或修正判断 |
