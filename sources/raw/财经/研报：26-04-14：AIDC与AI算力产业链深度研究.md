>
> 报告日期：2026年4月14日 | 数据截止：2026年4月13日

## 一、核心结论与产业主判断

AI 产业的竞争中心正在从**模型参数竞争**，全面转向**基础设施竞争**；从训练竞争转向推理与服务竞争；从卖硬件竞争转向平台化分成竞争。

- AIDC = 这个时代的物理底座
- 算力租赁 = 流量入口
- 存储和交换芯片 = 关键配套
- Token 成本 = 全球竞争力的本质指标

---

## 二、AIDC 产业链总框架

### 2.1 核心逻辑：一条清晰的需求传导链

AI 应用爆发 → 推理/训练需求激增 → 算力缺口扩大 → 大厂 Capex 增加、租赁需求爆发 → AIDC 建设启动 → 招标订单落地 → 电力/液冷/服务器/交换/光模块/存储全链受益

**典型项目：**
- 大同太行二期 187MW
- 芜湖算力中心 262MW
- 内蒙古和林格尔 110MW
- 市场已从零散小机房进入 **GW 级园区时代**

### 2.2 算力租赁 vs AIDC：本质区别

| | 算力租赁 | AIDC |
|------|----------|------|
| 性质 | 服务模式 | 物理基础设施 |
| 标的 | 租赁公司 | 数据中心建设与运营 |
| 关系 | 算力租赁公司是 AIDC 的**租户/客户** |

**大厂当前策略：自建/代建 AIDC 与算力租赁双头并进。** 先进模型依赖英伟达高端算力，合规性要求使大厂难以直接获取，第三方算力租赁公司有渠道。同时将一次性大额资本开支（H100 集群约 30 亿、B200 约 40 亿）转化为小额运营支出。

## 三、为什么 2026 年是建设拐点年：双轮驱动

### 第一轮：国产算力供给放量（基石）

GPU 放量不只是卖卡 → 配套服务器、电源、液冷、交换机、光模块、存储、土建、电力和运维一起交付 = **产业配套需求的乘数效应**

### 第二轮：H200 供给缓解（加速器）

此前大厂 Capex 并非没预算，而是高端算力受限，建设节奏被迫延后。H200 供应缓解 → 积压需求加速释放。

---

## 四、全球算力结构演变

### 4.1 美国：从训练中心 → 推理中心

AI 行业从「拼训练参数」进入「拼推理吞吐、调用成本、服务规模」阶段。Agent、多轮交互、多模态推理等新形态使单次任务消耗的推理算力大幅提升。商业化依赖的是调用频次、留存和场景嵌入深度，而非参数数量。

### 4.2 中国：国产推理算力爆发三阶段

| 阶段 | 特征 | 核心受益 |
|------|------|----------|
| 第一阶段 | 租赁和云端承接 | **算力租赁公司与云厂** |
| 第二阶段 | 行业化私有部署 | 垂直行业 |
| 第三阶段 | 生态繁荣 | 全产业链 |

> 不是一步到位全面替代，而是**分层突破**。短期最直接的受益在算力租赁与云厂层面。

## 五、算力通胀：指数级需求与线性供给的碰撞

### 核心判断

全球 AI 算力需求**指数级上升**，而供给仍然接近**线性增长**。两者之间的「算力通胀鸿沟」越来越大。

**只要需求增长斜率远大于供给增长斜率，产业链价格就会维持强势。** 这是判断算力租赁公司持续涨价能力的底层逻辑。

---

## 六、AI 算力需求持续爆发的三大驱动力

### 6.1 Token 消耗量指数级上升

- **2026 年 1-2 月**：国内 Token 总量**超越美国**，曲线快速抬升
- 2026 年以来周度 Token 消耗增加 **280%**
- 连续 5 周超过美国
- Token 消耗从少量开发者驱动 → 海量终端用户和企业场景共同驱动
- 多模态大模型、编程大模型的 Token 量对比传统文本推理**增长 10 倍以上**
- AI 从实验性工具 → 生产工具 → Token 消耗成为常态需求

### 6.2 OpenClaw 框架普及

- 传导路径：开源社区贡献 → OpenClaw 框架 → API 接口标准化 → 千行百业 API 调用 → 推理需求刚性拉动
- 过去模型调用零散定制化；未来标准化、API 封装后，需求像云计算 SaaS 调用一样被快速放大

### 6.3 国内互联网大厂订单翻倍

- 腾讯、阿里等头部客户近期订单同比增长**超过 100%**
- 采购以万卡集群为单位，租约 5 年起步
- 真实产业需求已进入**从验证到扩产**阶段
- 配套体系（AIDC、服务器、交换、存储、租赁）全线受益

## 七、中美 Token 成本对比与出海竞争力

**中国大模型 Token 成本仅为欧美 1/5 至 1/20。**

成本优势 = 动态扩张能力。谁能把推理成本打下来，谁就更有机会在全球市场抢占定价权和份额。

Token 出海趋势已显著：中国模型在 OpenClaw 的调用量排名持续靠前，连续 5 周超过美国。

## 八、算力租赁：进入卖方市场

### 涨价已成行业普遍动作

- 不是局部试探，是行业整体供需关系变化
- 服务商议价能力提升

- 马太效应显著，玩家持续减少
- 先发者拥有渠道、客户关系和规模三重壁垒

### 从固定收租 → Token 分成

| | 传统模式 | 新模式 |
|------|----------|--------|
| 收费方式 | 固定租金 | **Token 消耗量分成** |

## 十一、存储：被低估的 AI 基础设施核心受益方向

### 三大受益逻辑

1. 训练与推理需要更高内存带宽和更大容量配置
2. 多模态、Agent、企业级知识库调用显著提升存储密度需求
3. AIDC 建设必须配套高速存储与并行文件系统

> **存储板块不能只按传统消费电子逻辑来看，应纳入 AI 基础设施核心链条，按服务器端的高成长逻辑重新定价。**

## 十六、当前市场操作框架

### 指数环境

- 沪指临近上升趋势线突破关键位（约 4,009~4,011 点）
- 成交量持续低迷（2.3~2.5 万亿以下，历史上升趋势期均在 2.6 万亿以上）
- 中东地缘政治不确定性（4 月 22 日关键节点）压制资金情绪
- 短线板块效应分散，无明显资金聚集效应，多数标的属超跌反弹性质

### 核心操作原则

> 不见兔子不撒鹰——追求持续性而非最低点。
> 上升趋势确立后，任何位置买入都是赚钱的，早抄底或晚抄底对最终收益差异不大。

### 六大风险

| # | 风险 |
|---|------|
| 1 | 地缘政治：中东 4/22 节点若继续升级，全球风险偏好大幅下降 |
| 2 | b 浪陷阱：当前上涨可能仅为 b 浪反弹，成交量未放量前切勿激进建仓 |
| 3 | 标的位置：多数 AI 标的已从 3 月低点反弹至前高附近，可上可下 |
| 4 | 逻辑过时：若两周以上指数才突破，当前备选标的需全部重新评估 |
| 5 | 纯概念爆雷：拿不到订单、放不出业绩的算力租赁公司存在爆雷风险 |
| 6 | 算力通胀缓解：若 H200 大幅放量或国产算力超预期，租赁价格可能承压 |
