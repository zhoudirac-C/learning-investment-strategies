# 国产算力产业链与 Token 经济学深度研究报告——国产闭环生态、商业模式升级与投资机会

> 2026 年 5 月 | 行业深度 | 昇腾 950 超节点放量 + Token 工厂模式跃迁 + 中美算力共振

## 报告摘要与核心观点

国产算力产业链已迎来从硬件突破到生态闭环、再到商业模式升级的关键拐点。**2026 年新增算力市场国产卡占有率有望首次反超英伟达**，华为昇腾扛起半壁江山。

更值得关注的是商业模式从传统算力租赁向 **Token 工厂模式**跃迁：Token 分成能让头部运营商净利率从 10%-15% 抬升至 **30% 以上**，每卡收入提升至裸算力租赁的 **5-10 倍**。

三大投资主线：① 国产先进制程与封测设备材料受益昇腾 950 放量；② 散热、互联、连接器等基础设施迎确定性国内增量订单；③ 算力租赁向 Token 工厂转型的头部运营商。

| **价格下调** | 国产算力实现成本控制，打通需求侧逻辑 |
| **批量上市** | 中国半导体制造链条已基本走通，产能可保证、交付可稳定 |

### 1.2 2026 年关键预测

## 二、国产算力闭环生态：从卡点突破到全链路放量

### 2.1 昇腾 950 超节点：产业放量核心引擎

- 先进制造 + 先进封装协同推进，950 是国产设备第一大终端客户
- 高度自主可控，推动光刻、刻蚀、薄膜、量测等环节国产替代加速
- 4 月底年报一季报已印证：算力租赁业绩超预期、半导体材料集体超预期、前后道设备景气度上行

### 2.2 受益环节全景

| 产业链环节 | 增量逻辑 | 景气持续性 |
|-----------|---------|-----------|
| 先进制造设备 | 950 是国产设备第一大客户 | 3-5 年 |
| 先进封装 | CoWoS 等 2.5D/3D 封装，超节点对带宽指数级需求 | 3 年以上 |
| 半导体材料 | 靶材、光刻胶、电子特气国产替代率快速提升 | 中长期 |
| **散热** | 全液冷板、CDU，超节点功率密度倒逼 | 国内增量订单确定 |
| **互联** | 高速背板连接器、铜缆，8192 卡互联 | 受益最直接 |
| **光通信** | 全光互联、光模块、CPO，从海外链向国内链拓展 | 新增量 |
| 存储 | HBM、SSD，超节点对内存带宽诉求 | 结构性偏紧 |
| 电源 | 800V 高压、碳化硅 | 新一波成长机会 |

### 2.3 科创板的预期差

23-26 年间 A 股 AI 上涨主要是给英伟达/谷歌/亚马逊供货的海外链厂商。国产自主可控路径下的科创板此前相对冷清——科创 50 尚未回到 2020 年高点。随着超级周期向国内传导，科创板有望迎来从「被低估」到「价值重估」的窗口。

## 三、Token 工厂模式：商业模式的范式跃迁

### 3.1 概念对比

| 维度 | 传统算力租赁 | Token 工厂 |
|------|------------|-----------|
| 商业基因 | 重资产资源生意 | **价值附加生意** |
| 交付物 | 底层硬件物理控制权 | 标准化智能内容 Token |
| 计费模式 | 月固定租金，单卡 25-30 万/年 | **按 Token 调用量分成**，运营商获 60-80% |
| 利润率 | 净利率 10-15%，5 年回本 | 净利率有望 **30%+**，2 年回本 |
| 收入弹性 | 刚性上限 | Token 服务收入可达裸算力 **5-10 倍** |

### 3.2 盈利提升三大来源

1. **Token 调用分成杠杆效应：** 服务器折旧与电力为刚性固定成本，超额分成几乎全部转化纯利润
2. **精细化运营：** Token 工厂通过算力细切割按需收费，有效时长从 8 小时提升至 12 小时→效率再提升 50%
3. **价格机制灵活：** 收费标准为市场主流报价 7 折左右

### 3.3 落地节奏

某头部算力运营商已明确转型 Token 工厂：今年在五象云谷落地 4 个万卡集群，二期 AIDC 电力容量将达一期 **10 倍**，通过多种方式具备 300 亿资金规模支持算力业务。

> Token 分成、Token 工厂的落地将在 **2026 年二三季度**密集出现。

## 四、北美算力共振

### 4.1 涨价实锤

5 月 4 日北美 H100 和 B200 租赁价格分别较 4.30 上涨 **22%** 和 6%。谷歌 CEO 明确表示「算力受限、云收入本可更高」，预计未来 24 个月消化 50% 积压订单，需求能见度延伸至 **2028 年**。

### 4.2 头部玩家预期差

虽然服务器价格越来越高，但头部算力租赁玩家作为英伟达全球云服务合作伙伴，拿货价（出厂价）其实没有变。「无中间商赚差价」优势使头部运营商净利率远高于二线厂商。

## 五、风险因素与产业演进

### 5.1 短期风险

乐观预期已较充分、大厂可能联手控价、H200 可能逐步放开 + 国产先进制程产能翻倍、算力稀缺性会随技术演进而被优化。

### 5.2 中长期：从硬件高溢价到软件平台生态

凡是足够贵、足够重复、足够刚需的东西，最终都会被产业链优化。标志性事件：**大基金**（以前主投中芯、长存等硬件）正在领投 DeepSeek（估值逼近 450 亿美元），目的打通「中国设备 + 中国芯 + 中国模型」闭环。

### 5.3 三阶段投资框架

| 阶段 | 时间 | 核心矛盾 | 主线 |
|------|------|---------|------|
| 第一阶段 | 2026Q2-Q4 | 超节点放量、产能瓶颈 | 先进制程设备 + 封测 + 散热互联 |
| 第二阶段 | 2026H2-2027 | 商业模式重构 | Token 工厂运营商 + AIDC |
| 第三阶段 | 2027-2028+ | 生态闭环、标准定义 | 国产模型 + 平台 + 软件生态 |
