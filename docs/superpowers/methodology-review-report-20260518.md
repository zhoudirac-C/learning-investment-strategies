## 六、关键结论

1. **市场周期判断升级**：从单一"涨跌判断"升级为"四阶段周期+调整性质+修复性质"的完整方法论体系，已入 `market-cycle-framework.md`。

2. **活口分类体系完善**：劣性轮动识别标准补充入 `livestock-classification.md`，与退潮活口/复苏活口形成完整闭环。

3. **事件催化 vs 资金主动扩散**：这是本次 Review 最重要的方法论发现。区分消息驱动和资金主动选择，对结构性行情的参与节奏有决定性影响。

4. **分歧中找机会**：从口号变为可操作原则——"低位、有辨识度、有产业链位置"，最佳买点在分歧中，最佳卖点在一致时。

5. **调整≠行情结束**：5/15–5/18 的调整验证了"调仓换股窗口"框架。资金从高位泛科技向硬核方向迁移，核心逻辑未变。

6. **美股判断教训**：4/14 "非反转"判断被证伪，提醒在宏观拐点处（和平预期、AI 盈利前景）需要更灵活的框架，避免过度依赖技术性判断。

7. **油价判断教训**：地缘政治变量变化极快，"长期周期"判断容易被突发事件打断。博主后续将油价降级为"扰动因素"是正确的框架修正。
