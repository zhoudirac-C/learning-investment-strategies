# 选股框架

## 当前状态

已从 2025-12 至 2026-05 的全量财经 raw 中补充个股身份框架。选股前必须先判断标的在市场、主线和板块中的角色。

## 个股身份

| 身份 | 典型特征 | 处理方式 |
| --- | --- | --- |
| 核心龙头 | 主线辨识度最高，能带动板块 | 分歧低吸或持股跟踪 |
| 容量标 | 成交额大，机构和趋势资金能参与 | 用于确认主线强度 |
| 趋势票 | 基本面或产业逻辑持续兑现 | 拉长持股周期 |
| 补涨票 | 主线扩散后低位跟随 | 只做节奏，不恋战 |
| 后排跟风 | 无独立逻辑，靠情绪拉动 | 高潮期风险最大 |
| 退潮活口 | 分歧初期暂时抗跌 | 不追，观察是否补跌 |
| 复苏活口 | 充分分歧后率先修复并带动扩散 | 重点跟踪 |
| 案例票 | 用于解释逻辑，缺少持续跟踪条件 | 不自动纳入股票池 |

## 核心优先规则

退潮、混沌和强分歧阶段，优先买核心回调，避免追后排上涨。后排只有在主线扩散早期有交易价值；当主线完成全链条补涨后，后排追高风险会显著上升。

## 产业逻辑验证

个股不能只因为出现在题材中就被视为有效标的。需要检查：

- 是否处在主线核心环节。
- 是否有订单、业绩、涨价、产能、客户或技术验证。
- 是否是资金主动选择的核心或容量标。
- 是否存在逻辑证伪或事件兑现。

## 需要沉淀的问题

- 如何判断产业逻辑是否仍有效。
- 如何把博主历史提及转成可复盘的个股案例。
