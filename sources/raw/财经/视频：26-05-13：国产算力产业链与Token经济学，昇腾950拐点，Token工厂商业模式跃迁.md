## 前言：国产算力的第一个拐点

信号来自 **DeepSeek V4 与昇腾 950 超节点的组合**。DeepSeek V4 上线后价格说明中有一行小字：「受限于高端算力，目前 Pro 版本数据吞吐量非常有限，等到下半年昇腾 950 超节点批量上市后，Pro 价格有望大幅下调。」

三个关键词——**超节点、价格下调、批量上市**——分别对应国产算力产业化必须跨越的性能、经济性和交付三大门槛。

与此同时，需求端和商业模式也在发生根本变化：豆包已开始测试收费，海外算力价格持续上涨，**Token 工厂模式**正在重构算力服务的商业逻辑和估值体系。

**三条投资主线：**
1. 国产先进制程和封测设备材料
2. 散热、互联、连接器等 AI 基础设施环节
3. 算力租赁向 Token 工厂转型的头部运营商

## 一、昇腾 950 超节点：国产算力放量的核心引擎

### 三层意义

**第一层：解决国产 AI 卡单卡性能不足。**

国产 AI 芯片短期很难在单卡性能上追平英伟达最先进 GPU，这是产业现实。但 AI 计算不一定要只看单卡跑分——超节点的核心思想是用先进互联架构把大量算力卡组成新计算单元，**通过系统协同弥补单卡差距**。

> 单卡性能不能追平 → 就需要更多卡、更强互联、更高带宽、更低延迟、更强液冷、更复杂软件调度来实现系统级追赶。这意味着超节点会放大对整条基础设施链的需求。950 拉动的不只是芯片本身，而是封装、连接器、光模块、液冷、电源、存储、服务器和 AIDC 建设。

**第二层：对先进封装提出集中需求。**

国产 AI 芯片要放量，不能只靠设计端突破，必须有制造端、封装端和材料端共同支持。950 成为国产高端 AI 算力核心产品，就会成为国产先进设备和材料最重要的终端客户。

> 以前市场担心国产半导体设备和材料到底有没有足够的订单支撑。现在 950 进入批量交付，相当于给这些环节提供了明确的产业牵引。光刻胶、靶材、刻蚀设备、薄膜沉积设备、量测设备、封测设备、先进封装材料、基板、载板——从「国产替代概念」变为「直对终端需求」。

**第三层：高度自主可控。**

美国持续收紧高端芯片和半导体设备出口管制，中国高端 AI 算力不可能长期依赖外部供应。950 要真正量产，关键环节必须实现国产化——从设计、制造、封装、材料、设备、服务器、散热到软件生态，都需要自主可控链条。

> 过去国产半导体链条由政策推动，现在逐步变为**需求推动**。AI 算力是最确定的增量需求之一，而昇腾 950 就是把需求传导到国产供应链的核心载体。

### 产业链已出现验证信号

4 月年报和一季报披露情况：
- 算力租赁公司业绩超预期 → 下游算力需求真实存在
- 半导体材料（含靶材）业绩集体超预期 → 国产制造链条景气开始兑现
- 先进制程扩产带动前道设备需求上行，后道封测同样强势

> AI 放量不是孤立发生，而是推动前道、后道、材料和设备共同景气。尤其光刻机和光刻胶——美国最严芯片法案之后，对国产半导体的理解不能只停留在「刻」的环节，还要延伸到「光」的环节。

## 二、八大受益环节

950 采用全液冷散热、高速背板连接器和全光互联——这三个方向非常关键，说明 AI 算力竞争是系统工程竞争。

| 环节 | 逻辑 | 景气周期 |
|------|------|---------|
| **先进制造设备** | 光刻、刻蚀、薄膜沉积、量测检测，950 是国产设备最大终端客户 | 3~5 年高景气 |
| **先进封装** | Chiplet/2.5D/3D 封装，超节点对带宽要求指数级上升 | 3 年以上 |
| **半导体材料** | 靶材、光刻胶、电子特气、CMP 材料，从实验验证走向批量供货 | 中长期 |
| **散热** | 全液冷从可选变必须，冷板/CDU/液冷管路/泵/换热器/温控系统 | 国内增量订单确定 |
| **互联** | 高速背板连接器、铜缆、高速线缆交换模块、高速 PCB——8192 卡互联需求，受益最直接 | 增量最直接 |
| **光通信** | 从供海外链扩展到供国内链，全光互联/CPO/硅光/光交换 | 多元景气来源 |
| **存储** | HBM/SSD/内存/企业级存储，推理爆发推升存储带宽和容量需求 | 结构性偏紧 |
| **电源** | 800V 高压/碳化硅/UPS/机柜电源/配电系统，每瓦电都关系 Token 成本 | 新成长机会 |

## 三、Token 工厂：商业模式的范式跃迁

### 3.1 算力租赁 → Token 分成 → Token 工厂

| 维度 | 传统算力租赁 | Token 工厂 |
|------|------------|-----------|
| **商业基因** | 重资产资源生意 | **价值附加生意** |
| **交付物** | 底层硬件物理控制权 | 标准化智能内容 Token |
| **计费模式** | 月固定租金，单卡 20~30 万/年 | **按 Token 调用量分成**，运营商获 60~80% |
| **利润率** | 净利率 10~15%，3~5 年回本 | 净利率有望 **30%+**，2 年回本 |
| **收入弹性** | 刚性上限 | Token 服务收入可达裸算力 **5~10 倍** |

> 三者的关系：算力租赁是基础形态（赚固定租金），Token 分成是收入模式（赚调用量增长的钱），Token 工厂是能力体系（支撑从租赁升级到分成）。

### 3.2 盈利提升三大来源

**1. Token 调用分成的杠杆效应。** 服务器折旧、电力、机房、液冷、运维——大量固定成本。传统租赁下收入上限被锁死，Token 工厂下固定成本覆盖后额外分成大部分直接转化利润。调用越多，边际利润越高。**把算力运营商从按资产收费变成了按智能生产量收费。**

**2. 算力使用率提升。** 传统租赁整卡整月出租，客户用不用满不影响运营商收入。Token 工厂将算力细颗粒度切割调度，白天/晚上用户错峰，复杂/简单任务分层，3 小时/5 小时任务按需匹配。每日有效使用时数从 8 小时提升至 12 小时 → 设备利用率再提升 **50%**。

**3. 价格机制灵活。** 当前 Token 工厂收费为市场主流大模型报价的 7 折左右。未来即使 Token 价格下降 10~20%，但有效时长从 8 小时→12 小时提升 50%，总体仍可保持甚至提高收入。**靠价格、调用量、利用率、成本优化四个变量共同驱动。**

> 正常情况**两年多即可完全收回成本**，远优于传统租赁约五年回本周期。

### 3.3 需求端和供给端共同推动

**需求端：** 豆包月活 3.45 亿进入 C 端付费阶段（标准版 68 元/加强版 200 元/专业版 500 元），日均 Token 调用量超 120 万亿且过去三个月翻倍。C 端付费验证了重要事实：用户在高价值场景中愿意为更强能力买单。企业端金融、法律、医疗等场景 AI 替代高成本人力，付费意愿更强。

**供给端：** 英伟达提出 Token 工厂经济学，重塑全球数据中心建设方向——未来数据中心不只是存放服务器，而是专门生产 Token 的工厂。核心竞争指标从「有多少张卡」变为「每瓦电生产多少 Token」。

> MiniMax 的大模型价值公式「智能密度 × Token 吞吐量」反映同一方向：AI 产业正从模型参数竞争走向智能生产效率竞争。Token 工厂就是新阶段的基础设施形态。

### 3.4 产业落地进展

头部算力运营商已明确把 Token 工厂作为核心发展方向。典型模式：建设大规模 AIDC 和万卡集群，部署闭源模型，与模型公司基于 API 调用 Token 分成，运营商获约 **70%** Token 收入。

- 已规划在五象云谷落地 **4 个万卡集群**
- 二期 AIDC 电力容量达一期 **10 倍**
- 通过多种方式准备约 **300 亿元**资金支撑算力业务

> Token 分成和 Token 工厂密集落地预计从 **2026 年 Q2-Q3** 开始出现。单一龙头率先验证，随后扩展到云厂商和算力服务厂商整体转型。

### 3.5 如何验证真正的 Token 工厂

1. 是否有模型公司部署在自有算力中心
2. 是否有基于 API 调用的 Token 分成协议
3. Token 收入占比是否提升、设备有效使用时长是否提升、净利率是否从 15% 向 30% 靠拢
4. 是否具备软件优化和平台调度能力
5. 是否有足够低的电力成本和稳定的 AIDC 资源

## 四、中美算力共振

### 4.1 北美涨价实锤

5 月 4 日北美 H100 和 B200 租赁价格分别较 4.30 上涨 **22%** 和 **6%**。谷歌 CEO 明确表示「谷歌云收入受算力限制，如果算力更多收入本可以更高」，预计未来 24 个月消化约 50% 积压订单，需求能见度至少延伸至 **2028 年**。

黄牛市场验证供需失衡：B300 现货从 4 月中旬 545 万元 → 4.30 涨至 643 万元 → 五一假期末突破 **700 万元**。客户等不及正式交付周期，愿意支付溢价抢现货。

### 4.2 三条传导链

| 链条 | 逻辑 |
|------|------|
| **存储链** | 涨价+满产满销，「从头发丝到后脚跟全都缺」 |
| **CPU 链** | 服务器 CPU 受 AI 推理拉动结构性供给紧张，涨价幅度可观 |
| **算力租赁链** | 需求确定性最强，叠加出口管制变相收紧供给 |

### 4.3 头部玩家的关键预期差

现货服务器价格越来越高，但**头部算力租赁玩家的出厂价其实没有变**。头部玩家本就是英伟达全球云服务合作伙伴，可直接或优先采购，不需要去黄牛市场抢 700 万一台的 B300。

> 二线厂商高价抢货 → 成本端大幅上升 → 即使租赁价格上涨利润也被吞噬。头部玩家出厂价稳定、租赁价格上涨 → **利润弹性极大**。这就是「无中间商赚差价」的优势。

## 五、风险提示

1. **乐观预期已较充分：** 部分公司股价已包含较多未来预期，服务器 700 万一台已接近大厂测算盈亏平衡线
2. **控价、举报、合规风险：** 大厂可能联手控价，非合规渠道可能被收紧
3. **供给侧改善：** H200 可能逐步放开 + 8~9 月国产先进制程产能翻倍
4. **线性外推风险：** 算力稀缺不等于永远稀缺。AI 产业基本规律——凡是足够贵、足够重复、足够刚需的东西，最终会被产业链不断优化（模型压缩、量化、蒸馏、稀疏计算、编译器优化、自研 ASIC、边缘部署）

## 六、三阶段投资框架

| 阶段 | 时间 | 核心矛盾 | 投资主线 |
|------|------|---------|---------|
| **第一阶段** | 2026Q2-Q4 | 超节点放量、产能瓶颈 | 先进制程设备 + 封测 + 散热互联 |
| **第二阶段** | 2026H2-2027 | 商业模式重构 | Token 工厂运营商 + AIDC + 算力租赁 |
| **第三阶段** | 2027-2028+ | 生态闭环、标准定义 | 国产模型 + 平台 + 软件生态 |

> 短期看硬件放量，中期看商业模式升级，长期看软件平台和生态标准。Token 经济学正是这场产业范式转变的核心标志。
