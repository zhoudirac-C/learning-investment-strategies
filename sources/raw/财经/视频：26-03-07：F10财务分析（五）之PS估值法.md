# F10财务分析（五）之PS估值法

> 2026年3月7日 | F10系列第五课 | 估值体系第三讲

---

## 一、PE 什么时候失效

| 情景 | 案例 | 问题 |
|------|------|------|
| 大幅亏损 | 寒武纪 2022-2024 净利润为负 | PE 无法计算 |
| 利润极度微薄 | 成长期科技公司净利润≈0 | PE 上千倍失去比较意义 |
| 一次性利润扭曲 | 政府补贴/资产出售 | 当期利润不反映真实经营 |
| 密集研发 | 早期 AI/生物医药 | 大量研发投入压低利润 |
| 商业模式重构 | 互联网平台初期 | 亏损是战略选择（获取用户） |

**此时用 PS（市销率）。**

---

## 二、PS 公式与核心变量

```
PS = 总市值 ÷ 营业收入 = 股价 ÷ 每股营收
```

PS=5 → 市场愿花 5 元买 1 元营收。PS=100 → 花 100 元买 1 元营收。

**PS 的核心弱点：营收本身 ≠ 价值。** 必须配合**毛利率**和**增速**。

### 毛利率 + 增速 → PS 基准区间

| 毛利 | 增速 | PS 区间 | 代表行业 |
|------|------|---------|----------|
| > 80% | > 100% | **50-100x** | AI 软件/大型 API |
| ~60% | ~50% | **20-40x** | 半导体设计/高端医疗器械 |
| ~40% | ~20% | **6-12x** | 硬件厂商 |
| ~20% | ~10% | 低区间 | 组装制造/电商/零售 |

## 三、量能参数

2.5 万亿流动性基准下，PS 容忍度系统性提升：

| 赛道 | PS 区间 |
|------|---------|
| AI 算力芯片（寒武纪） | 50-120x |
| AI 应用/大模型 | 8-25x |
| 创新药 | 10-40x |
| 云计算 | 5-15x |
| 互联网平台 | 4-10x |

**对标：** 英伟达 2025 年 PS ≈ 29x（成熟盈利期顶级 AI 芯片的标杆）。

## 四、案例一：寒武纪——从 PS 到市概率

### 关键数据

| 时间 | 市值 | 营收 | PS |
|------|------|------|-----|
| 2022 | 200 亿 | 7.29 亿 | 14x |
| 2025 峰值 | **5800 亿** | ~57 亿 | **111x** |
| 当前 | 回落 | — | 稀缺性溢价被压缩 |

**市值暴涨 29 倍，营收只涨 8 倍 → 核心是估值膨胀，不是业绩增长。**

### 市概率（Alternative Model）

当 PE 无法用、PS 也失真时，资本市场用「成为龙头的概率」定价。

```
隐含市值 = 龙头市值 × 成功概率

英伟达市值 ≈ 20 万亿 RMB
寒武纪 6000 亿峰值 ≈ 20 万亿 × 3%
```

| 概率 | 隐含市值 |
|------|----------|
| 5% | 1 万亿 |
| 3% | 6000 亿（2025 峰值对应） |
| 1% | 2000 亿 |

**摩尔线程、沐曦股份上市后 → 寒武纪稀缺性溢价被快速压缩 → PS 向行业均值回归。** 这也是寒武纪近期表现不佳的核心原因。

### 估值工具可靠性排序

| 工具 | 可靠性 | 依赖 |
|------|--------|------|
| **PE** | 最高 | 基本面确定，当期利润倍数 |
| **PS** | 中等 | 增速维持，毛利率不恶化 |
| **市概率** | **最低** | 全靠预期，叙事破裂即崩溃 |

## 五、案例二：科大讯飞——成熟 PS 案例

| 指标 | 数据 |
|------|------|
| 营收 | 233 亿 |
| 研发费用 | 38.92 亿（~20%） |
| 净利润 | 5.6 亿（净利率 2%） |
| 市值 | 1200 亿 |
| PS | **4-6x** |
| 毛利 | 40%（AI 软件高+硬件低混合） |
| 增速 | 19%（中速，非高速） |

### PS 天花板推算

```
PS 天花板 = 目标净利率 × 合理PE ÷ 当前净利率
= 12% × 30 ÷ 2% = 180x
```

但需打折：时间折现 + 竞争不确定性 + 中速增长 → **打 0.5-1 折 = 10-15x（乐观预期）**。

**当前 4-6x = 市场给的保守定价。** 投资者赌的是研发兑现、净利率从 2%→12%、PS 从 4-6x 修复到 10-15x。

## 六、案例三：芯源微——半导体设备 PS 实战

| 指标 | 数据 |
|------|------|
| 2025 年营收 | 19.48 亿（+11.11%） |
| 市值 | 382.7 亿 |
| 静态 PS | **19.64x** |
| 毛利 | 34.52%（涂胶显影~40%，后道封装~30%） |

**对标：** 北方华创（毛利 ~40%，PS 8-12x）vs 拓荆科技（毛利 ~45%，PS 15-20x）。芯源微 19.64x 略高于行业合理区间。

### 前瞻 PS

| 年份 | 预计营收 | PS |
|------|----------|-----|
| 2026E | 28 亿 | **13.67x** |
| 2027E | 35 亿 | **10.93x** |

- 合同负债 8.03 亿（+70%），在手订单充足
- 机构预测 2026 年营收增速超 40%
- 市值从年初 200 亿→382 亿接近翻倍，但营收只增 11% → **本质上已是 PS 从 11x→19.6x 的提前定价**

**风险：** 若 2026 订单转换不及预期、营收增速未达 40% → PS 从 20x 回调到 10-12x。

### 芯源微的核心壁垒

**国内唯一实现前道涂胶显影设备量产的厂商。** 该设备此前被东京电子垄断，国产替代率不足 10%。赛道确定性极高 → PS 溢价高于科大讯飞。

## 七、PS 四大误区

| 误区 | 正确认知 |
|------|----------|
| **PS 越低越好** | 毛利 5% 的电商 PS=1x vs 毛利 80% 的软件 PS=20x，后者反而更便宜。同毛利率同增速下越低越好 |
| **用 PS 给低毛利行业估值** | 批发商/超市毛利 2%，PS 0.x 倍 ≠ 便宜，净利率可能为负 |
| **无视增速可持续性** | 高 PS 建立在假设的高速增长上，一旦不及预期泡沫破裂（寒武纪 2025.1.16 单日跌 14.65%） |
| **低 PS+低 PE=便宜** | 低增速+行业衰退的低 PS/PE = 价值陷阱，不是机会 |

---

## 八、PS 的定位：过渡期工具

**PS 的核心定位：** 当公司从亏损走向盈利，从高成长走向稳定 → PS 逐步让位于 PE。

用 PS 买入时必须有**清晰的 PE 接力时间表：** 这家公司什么时候盈利？什么时候能用 PE 重新定价？没有这个时间定义，PS 只是猜测工具。
