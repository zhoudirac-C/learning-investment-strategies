# 交易纪律

## 强分歧次日纪律

只做确认，不做盲目追高。

## 低吸主线条件（需同时满足）

1. 沪指守住关键位（如4150点）
2. 成交额缩到合理区间（如2.7-3万亿）
3. 跌停家数明显下降
4. 科技主线出现龙头领涨和板块普涨

## 降低仓位信号

1. 关键位被有效跌破
2. 成交额继续放大
3. 科技主线没有回流
4. 防御方向继续强于进攻方向

## 不能追的反弹

早盘指数反弹但成交额过大，科技板块只有少数个股脉冲而板块整体没有跟随——这种反弹更像是高位分歧后的技术性反抽，不是新一轮上攻确认。

## 反弹初期做T纪律

左侧止跌后的第二天，如果市场整体兑现幅度不大、接力中规中矩但不弱，可以继续持股或做 T。

但若回流仍是轮动式反弹，不是单一主线强一致加速，则不追高加仓。轮动反弹中，今天弱的方向后续也可能重新转强，擅长做 T 的可以围绕持仓打利润垫，而不是卖弱追强。

## 科创虹吸纪律

科创50或半导体单独加速时，不能只看指数涨跌。必须同时看全 A 上涨家数、上证是否站稳关键位、半导体内部梯队是否扩散。

若科创继续单独加速、全 A 仍有3000只以上下跌、成交额无法重新站上3万亿，则按虹吸型上涨处理：

- 持有核心，不追后排
- 日内做 T 降成本
- 回避纯补涨和故事票
- 等分歧低吸，不在一致加速时提高仓位

## B浪反弹操作纪律

### 买阴不买阳

B浪反弹空间有限，追高极易被套。只能在阴线/低开时建仓，绝不在阳线/冲高时追入。

### 完整吃完再减仓

该规则已被5月21日后的节奏上修：若指数一步到位打到原目标区间并重新修正结构，不能机械等待完整B浪，而要按新的结构和事件节点减仓。

### 炸板次日阴线买点（条件严格）

- **适用场景**：涨停炸板收出阴线或上影线的个股（如兆易创新）
- **买点条件**：① 当日确认为行情拐点；② 次日再收阴线
- **风险**：若当日不是拐点、次日继续下杀，则存在被套风险
- **核心前提**：当日最低点必须成立

## 产业链联动选股纪律

- 下游客户调整 → 上游供应商同步承压（如三大运营商调整→润建股份风险）
- 海外龙头企稳 → 国内关联标的机会（如英特尔上涨→中国长城联动）

## 分歧日趋势去留纪律

分歧日不是简单全卖或全留，而是先判断方向是否还在趋势内。

| 类型 | 纪律 |
|------|------|
| 破趋势方向 | 反抽按弱修复处理，不能恋战 |
| 趋势还在且逻辑硬 | 多等一等，看分歧后是否资金回流 |
| 远期题材补涨 | 只看承接，不用主线容错率保护 |
| 支线一致高开 | 等换手后看容量锚表现，不追公告情绪 |

尾盘低吸必须是条件单：只有全天单边下跌且接近2点55仍未修复时，才考虑尾盘低吸；若盘中已经反弹，则放弃，不把低吸变成追高。

## 结构动态修正纪律

技术结构必须服从实际走势。当指数直接一步到位打到原预判目标区间，且原有 B/C 结构不再对称时，应重新合并 K 线、修正波浪级别和目标位，而不是继续硬套旧 ABC 路径。

执行要求：

- 原点位到达但结构不匹配时，先重判结构，再决定仓位。
- 目标从4200上修到4700这类大调整，必须对应新结构，不是简单情绪乐观。
- 即便结构上修，仍遵守买阴不买阳、卖阳不追高。
- 反弹到关键位或事件节点时，顺势减仓，而不是把看多变成无条件持有。

## 正宗逻辑优先

小级别下降趋势中，优先选择调整充分、产业逻辑正宗、位置相对低的科技品种。蹭热点标的即便短线有弹性，也容易在分歧中出现大幅回撤，应回避。

## 二次冰点低吸纪律

二次冰点不是“低开就买”，而是等恐慌释放后的主动修复。

低吸条件：

1. 开盘先下探，最好是 1%-2% 的急杀。
2. 10 点前出现放量拉升。
3. 率先翻红的是机器人、PCB、先进封装等主动修复方向。
4. 跌停数量不再继续扩散。
5. 分时均线拐头向上，并放量突破日内高点。

禁止条件：

- 直接高开缩量冲高。
- 低开后弱反弹，午后继续创新低。
- 科技主线修复失败，只剩防御板块补涨。
- 当日涨幅已经超过 5% 后再追。

如果继续放量下杀且跌停数量超过 50 只，应放弃低吸，等待下一次明确止跌信号。

## 先买后卖做T确认纪律

极致冰点后的修复日，可以围绕已关注方向做先买后卖的 T，但买点必须等确认。

适合做 T 的条件：

1. 个股没有破趋势。
2. 属于 AI、半导体、存储、算力等主线硬逻辑。
3. 早盘跟随指数下探后，分时出现放量拐头。
4. 后一个反弹峰超过前一个反弹峰。

不适合做 T：

1. 个股已经放量破位。
2. 板块主线明显退潮。
3. 早盘只是弱反抽，没有放量突破前高。
4. 仓位过重，无法承受继续回踩。

日内节奏上，9:30-9:45 观察，9:45-10:30 等急跌确认，10:30-13:30 看主线修复质量并做 T，13:30 后进入风险窗口，原则上只卖不买。

## 假摔确认后的持股纪律

当快速下杀被进一步定性为假摔，且趋势核心没有集体破位时，纪律从“立刻判断修复成败”转为“持股等待修复验证”。

执行要点：

1. 趋势仍在、逻辑仍硬、没有明显破位的核心票，不在分歧低点恐慌卖出。
2. 盘中没有立刻修复，不等于假摔判断失效；上午不修复可等下午，下午不修复可等后续交易日。
3. 仍需观察趋势股能否维持新高或强承接，若强趋势股反而集体破位，则需要重新降低预期。
4. 该纪律只适用于趋势内核心，不适用于已经放量破位的后排和远期题材补涨。

## 盘中观点连续性纪律

盘中补发或午盘更新，首先要判断它是在提出新逻辑，还是在确认早盘观点的连续性。

若午盘只是确认早盘策略仍有效，则不因为盘中波动随意改变判断。执行上看五个连续性信号：

1. 指数没有继续放量杀跌。
2. 科技主线核心没有进一步破位。
3. 趋势股维持强承接。
4. 修复方向开始从局部扩散。
5. 早盘假摔逻辑没有被破坏。

若这些条件持续成立，盘面超预期时不急着兑现，继续持股比频繁做反向操作更符合原计划。若条件被破坏，再回到降预期和去弱留强。

## 退潮尾声的追高过滤

5月24日动态和周复盘共同补充了退潮尾声阶段的追高过滤规则。

当市场已经从极致分歧进入缩量修复，但高位连板和前期强势品种赚钱效应尚未完全修复时，不能把单日普涨视为新周期确认。

执行规则：

1. 周末充分发酵的 PCB、MLCC、材料股，周一若一致高开，不追高。
2. 先看是否放量承接；若高开后无量或快速回落，按兑现压力处理。
3. 等第一次分歧，看容量核心和主线内部能否承接。
4. 低位补涨只按轮动处理，不能用核心主线的容错率保护。
5. 半导体个股若发布减持公告，不用板块逻辑硬抗个股利空。
6. 退潮到穿越期，博弈穿越胜率低，优先选择买阴不买阳和赔率更好的低位方向。

这条纪律的重点不是否定主线，而是防止在主线中后段扩散时追在情绪高潮。

## 高开验证纪律

5月25日早盘进一步把高开验证拆成五个维度：

| 维度 | 有效信号 |
|------|----------|
| 指数 | 放量突破5日均线，用两天上涨抵消前一根大阴线 |
| 情绪 | 一进二晋级达到15家以上 |
| AI硬件 | PCB、MLCC、ABF、玻璃基板核心同步走强 |
| 容量核心 | 千亿市值或核心趋势票保持强势 |
| 国产算力 | 超聚变映射与海外AI硬件链共振，而非单日脉冲 |

若这些条件缺失，高开更可能是消息刺激后的冲高回落，不作为追高理由。

## 风格切换与突发利好过滤

5月25日盘中补发进一步说明：风格切换和突发利好同时出现时，不能只看当天涨跌。

执行纪律：

1. 当老登股补涨、科技承接转弱时，先保护科技利润，不在承接变弱后继续扩大进攻。
2. 老登股属于提前埋伏方向，涨起来以后不追。
3. 半导体这类突发利好发酵，若本来有一致性过高和减持压力，仍需看次日承接。
4. 远期技术催化兑现周期越长，越要降低当天加速后的追高冲动。
5. 大盘未稳稳站上4170前，不能把盘中加速简单解释为全面转强。

该纪律与“高开验证纪律”一致：消息改变节奏，但承接决定质量。

## 新技术路线发布后的追高过滤

5月26日韬定律视频强化了一个交易纪律：新技术路线可以改变中长期产业权重，但不能自动等价为短线买点。

过滤步骤：

1. 先区分“官方路线发布”“供应链订单”“业绩释放”三个验证层级，越靠前越偏预期，越靠后越能支撑趋势。
2. 若热门方向已经充分炒作，优先找未被充分挖掘但与路线强相关的细分环节，而不是追最热标签。
3. 对远期路线如3D折叠、封装内光学、High One，要降低当天涨幅的参考权重，提高后续验证权重。
4. 若板块只是题材扩散但核心容量票不强，按事件催化处理；若核心容量票、产业链上下游和指数环境共振，才可提高仓位弹性。

## 主升第二日的核心与后排

5月25日动态复盘把 AI 科技定位为主升第二日，同时也提示了主升第二日的分化纪律：核心方向可以继续看承接，后排和低位补涨不能无差别追。

执行规则：

1. PCB、半导体、光通信、国产算力等主线内部，优先看中军和前排人气标的，不用后排数量判断主线强弱。
2. 柜内 Rubin 后排弱、柜外光通信强时，说明资金在主线内部做高低切和预期差切换，不能追弱势后排。
3. 电力、功率器件、变压器等低位补涨方向集体启动后，若没有先手，按补涨尾声处理。
4. 微盘股与科技背离在主升初期可接受，但进入中后期若持续背离，要降低仓位而不是继续加杠杆。

## 强催化一致高开与防御对冲

5月26日早盘补充进一步把强催化日的执行纪律拆清楚：

1. 韬定律这类强叙事如果开盘过于一致，不能盲目顶一字板，先看换手、回封和20cm一进二。
2. 当日最强主线必须通过相对强度验证，例如国产半导体链要明显强于Rubin链。
3. Rubin链短期不证伪，但只做分歧后的核心低吸，不追后排一致高开。
4. 电力、煤炭等防御题材只在科技不及预期时作为对冲，不在科技核心承接良好时追涨切换。
5. 成交额回到3万亿以上不代表短线更简单，仍要只做强度确认后的前排。

## 普涨后分歧的持股与抄底窗口

5月26日两条盘中补发把“强主线分歧日”的执行节奏补全：

1. 昨日普涨后，今日调整属于正常现象，半导体只要不大跌且有承接，就先继续持股观察。
2. 真正强弱确认点放到次日，不因上午分歧过早否定主线。
3. 11:05后半小时到一小时若分歧释放充分，可以准备低吸。
4. 低吸对象不是跌幅榜，而是“跌得多 + 自己看好 + 属于近期主线核心 + 有分时承接”的标的。
5. 跌停、贴低点无拉回、后排弱分支、破位不能收回的票，即使跌幅大也不适用。

这条纪律和“不追一致”是一组：一致高开时不追，分歧释放后在看好的核心里找赔率。

## 高位压异动不是抄底对象

5月26日午盘补发用利通电子给“准备抄底”补了一个反向边界：同一天上午可以在主线核心里找分歧低吸，但高位强势票如果走势不健康、又接近异动监管压力，就不能继续按抄底框架处理。

执行规则：

1. “抄底”只适用于跌幅释放后的看好主线核心，不适用于高位不健康票。
2. 如果主要矛盾变成短线涨幅、压异动和兑现压力，买点要自动降级为卖点观察。
3. 已有利润的持仓，遇到高位不健康走势，应优先考虑利润保护和卖点识别。
4. 没有先手的人，不能因为盘中说过“可以抄底”，就去追高位强势票。

这条纪律把“低吸”和“让利润奔跑”区分开：低吸解决的是赔率和承接，利润奔跑解决的是已有仓位的兑现节奏，两者不能混用。

## 发酵日不追纪律

5月27日动态补充对"发酵日"给出了明确的操作边界：当产业链方向已经经过盘中发酵、市场已经充分反应时，即使逻辑正确，也不应在当日追高。

执行规则：

1. 新方向（如SST/功率器件）首次发酵日，记录逻辑和标的，但不追。
2. 真正的需求爆发窗口（如2026年下半年）才是布局时机。
3. "预热"和"发酵"是两个阶段：预热期适合研究和跟踪，发酵期适合持有先手，爆发后适合兑现或等回调。
4. 对于功率器件这类有明确时间节点的方向，先记好标的和时间节点，等回调或预热结束后布局。

这条纪律与"不追一致"是同一组规则在不同阶段的应用：一致高开不追、发酵日不追、利好兑现不追。
