# 国产CPU/GPU与软件

## 核心定位

5月21日晚间复盘中，国产 CPU、部分 GPU 和软件被列为半导体反弹减仓后的潜在资金承接方向。

其逻辑不是脱离科技主线另起炉灶，而是半导体核心方向涨高或进入事件减仓窗口后，资金继续在科技硬件链内部寻找低位、估值相对不高、有产业逻辑的分支。

## 交易逻辑

| 维度 | 判断 |
|------|------|
| 位置 | 相比部分半导体核心，估值和位置相对不高 |
| 产业属性 | 属于国产科技硬件和基础软件链条 |
| 资金来源 | 半导体其他方向减仓或高低切后的承接资金 |
| 交易阶段 | 半导体反弹减仓后观察承接，不在一致高潮时追 |

## 与半导体的关系

半导体仍是当前核心主线，国产 CPU/GPU/软件更多是后续切换备选方向。只有当半导体核心减仓后，这些方向出现板块级承接，才可上修为新一轮科技内部主攻方向。

## 观察标准

1. 半导体减仓资金是否回流国产 CPU/GPU/软件。
2. 是否出现容量中军和前排弹性标的同时走强。
3. 是否有国产替代、订单、生态适配或政策催化配合。
4. 是否与 AI 硬件、国产算力、信创软件形成同向扩散。

## 核心标的分层

| 层级 | 标的 | 定位 | 观察重点 |
|------|------|------|----------|
| 国产AI芯片容量核心 | 寒武纪 | 国产 AI 芯片核心，科创容量锚 | 订单、收入增速、合同负债、现金流和科创资金承接 |
| 国产CPU/DCU核心 | 海光信息 | 国产 CPU + DCU，兼顾通算和智算 | 金融、电信、能源等关键行业迁移，AI CPU/GPU需求和业绩兑现 |
| 国产GPU核心 | 摩尔线程-U、沐曦股份-U | 国产 GPU 双雄，资本化后稀缺性提升 | 出货、客户、生态、亏损收窄和估值消化 |
| 科创AI芯片弹性 | 芯原股份、翱捷科技、云天励飞 | 国产 AI 芯片设计弹性方向 | 是否有订单、模型适配、终端应用和业绩拐点 |
| 国产CPU与信创硬件 | 龙芯中科、中国长城 | CPU涨价、信创设备改造和国产替代 | 龙芯生态、央国企设备改造、中国长城是否成为信创先锋 |
| 服务器与整机配套 | 中科曙光、浪潮信息、紫光股份、中兴通讯 | 国产算力硬件和服务器配套 | 国产算力订单、整机交付、液冷/网络/存储配套 |
| 昇腾/CANN软件生态 | 润和软件、东方国信、软通动力、科大讯飞 | CANN 适配、AI 应用开发和模型迁移服务 | 是否从题材适配走向真实项目、收入和客户案例 |
| 国产数据库 | 达梦数据、海量数据、星环科技 | 基础软件替代，不是短线硬件主线 | 信创目录、金融能源客户、数据库迁移订单 |
| 国产中间件 | 东方通、普元信息 | 中间件国产替代 | 政企客户、信创招标和业绩弹性 |
| 操作系统平台 | 中国软件、中国长城、软通动力 | 麒麟、统信等国产操作系统生态映射 | 央国企替代节奏、生态兼容和政策招标 |
| AI应用/营销软件 | 蓝色光标、金山办公等 | 低位轮动和应用端弹性 | 只按轮动看，不作为硬件主线长期底仓 |

本地来源主要来自 2026-05-10 周复盘、2026-05-11 周末复盘补充、2026-05-12 股票池汇总、2026-04-24 国产算力生态视频和国产算力深度研究。外部信息更新后，摩尔线程-U、沐曦股份-U已可作为上市国产 GPU 方向跟踪，但仍要按业绩兑现、亏损收窄和估值消化验证。

软件方向需要降一档处理：UP 在 5 月 10 日周复盘里明确提示，国产软件可以吃资金轮动，但“不能长期拿”。因此软件池更多用于半导体减仓后的承接观察，不等同于核心硬件底仓。

## 2026-05-25：超聚变IPO与国产算力闭环观察

5月25日早盘把超聚变创业板IPO获受理作为国产算力板块催化，并把 DeepSeek 降价、昇腾推理芯片和 CANN 框架迁移纳入国产模型、国产芯片、国产框架闭环观察。

学习处理上，这些具体事实和参数需要二次核验；交易框架先保留两个判断：

1. 超聚变相关映射不能只看单日概念股冲高，要看国产服务器、液冷服务器、算力基础设施和服务商/经销商链条是否形成板块承接。
2. 国产算力若能与英伟达海外AI硬件链共同走强，说明资金认可“海外硬件景气 + 国产闭环替代”双线；若只剩映射脉冲，则持续性降低。

新增观察池：

| 方向 | 观察方式 |
|------|----------|
| 国产服务器整机 | 超聚变IPO进展、服务器订单、液冷服务器份额 |
| 国产AI芯片 | 昇腾、寒武纪、国产GPU出货与生态适配 |
| 国产框架/软件 | CANN适配、模型迁移、开发者生态和真实项目收入 |
| 服务商/经销商映射 | 只作为弹性映射，需看合作稳定性和业绩占比 |

## 2026-05-26：韬定律下的昇腾与AI系统闭环

5月26日视频把韬定律的最大工程资源倾斜点指向AI。原因是AI工作负载的吞吐能力可以直接转化为经济价值，华为未来更可能把资源集中到昇腾、HBM、光通信、超节点系统等方向。

对国产算力观察池的新增含义：

1. 昇腾不只按单颗AI芯片看，还要按系统级互联、HBM、CANN生态、超节点和整机交付一起看。
2. 数据移动成为瓶颈后，国产算力的关键验证点不只是算力峰值，也包括内存带宽、互联效率和软件适配。
3. 若后续昇腾路线从2.5D Chiplet走向3D折叠，先进封装和光通信会成为国产算力链的重要外溢方向。
4. 具体流片进展、型号规划和性能数据属于视频转述，暂以低置信度跟踪，等待官方、供应链或业绩侧验证。

交易上，国产算力仍不能只看概念映射。强信号是昇腾链、服务器整机、CANN适配、光通信和先进封装形成同向扩散；弱信号是只剩单日小票映射冲高。

## 2026-05-25 动态复盘：昇腾950超节点预期

5月25日动态复盘把国产算力走强归因到 DeepSeek 降价和昇腾950超节点落地预期。相比早盘的超聚变IPO映射，盘后视角更强调国产算力闭环中的“模型降价 + 昇腾超节点 + 国内算力厂商跟进”。

学习处理上，该方向需要拆成三层验证：

| 层级 | 验证点 |
|------|--------|
| 事件催化 | DeepSeek降价、昇腾950超节点预期、相关发布节奏 |
| 产业闭环 | 昇腾芯片、服务器整机、CANN框架、模型迁移和国产应用开发 |
| 订单兑现 | 算力集群交付、客户订单、收入确认和真实利用率 |

若国产算力与英伟达海外AI硬件链共同走强，说明资金认可“双线景气”；若只剩小票映射，则仍按短线题材处理。
