# 国家安全下的AI监管升级

> 2026年4月27日 | 专题视频 | Manus收购被叫停的深层含义

---

## 一、核心事件：Manus 被叫停

国家发改委禁止外资收购 Manus 并**要求撤销交易**——不是补充材料、不是延后审批、不是附加条件通过，而是**直接否定交易本身**。

这标志着 AI agent 在当前环境中不再是普通商业应用，而是被国家直接纳入**安全和战略科技资产**范畴。

### 时间线

| 时间 | 事件 |
|------|------|
| 2025年3月6日 | Manus 发布，AI agent 概念爆发（酷特智能等 20cm 涨停） |
| 2025年4月 | Benchmark（谷歌旗下风投）投资 7500 万美元，估值 5 亿美元 |
| 2025年7月 | Manus 关闭中国办公室，整体迁至新加坡 |
| 2026年初 | Meta 以约 20 亿美元收购（Manus 年收入约 1 亿美元） |
| 2026年4月 | **发改委叫停，要求撤销交易** |

## 二、监管逻辑的根本转变

**过去：** 看法律实体注册在哪里。创业公司用开曼/新加坡/香港架构做融资和上市是成熟路径。

**现在：** 看**技术来源和资产实质联系**：
- 核心团队是否来自中国
- 技术积累是否在中国形成
- 知识产权、工程经验、产品原型是否与中国生态高度相关
- 交易完成后控制权、技术、数据能力是否与海外科技巨头相关

**即使主体已迁至海外、交易发生在境外、员工已前往新加坡，仍被纳入审查。** 核心人员早在 3 月已被限制出境。

## 三、中美科技战三个阶段

| 阶段 | 竞争领域 | 手段 |
|------|----------|------|
| **第一阶段** | 产品与市场 | 商业竞争 |
| **第二阶段** | 芯片与供应链 | 出口管制（GPU/半导体/光刻机） |
| **第三阶段（当前）** | **科技资产所有权与资本** | 投资管控 + 资产归属争夺 |

**数据佐证：** 美国投资者参与中国科技投资从 2018 年的 412 亿美元降至 2025 年仅 3 亿美元（不足 1%）。

**双向管控：** 美国限制硅谷风投投资中国半导体/AI/量子计算；中国也开始反制，管控中国 AI 技术团队和应用资产流向美国。

**不止 Manus 一家：** 月之暗面（Kimi）、Stamfun、字节跳动均被监管机构提醒——未经批准不能接受美国资金或向美国投资者出售股权。

## 四、为什么是现在：国产承接能力已成型

DeepSeek V4 + 华为昇腾的出现改变了全球对中国 AI 能力的认知：
- **模型层：** DeepSeek 证明中国团队可用更高效率、更低成本做出全球竞争力大模型
- **应用层：** Manus 证明中国团队能做 AI agent 产品

如果这些能力被海外巨头吸收，中国将失去未来 AI 应用中的关键资产。但如果不让外流，就必须有自己的承接体系。

---

## 五、中国 AI 产业闭环的四根支柱

| 支柱 | 内容 | 代表方向 |
|------|------|----------|
| **国产算力** | Agent 运行需要持续推理算力（多轮推理+工具调用），非低成本单次问答 | 昇腾、国产AI服务器、国产云、AIDC |
| **国产模型生态** | Agent 依赖底层模型，国内大模型能力越强越不依赖海外 | DeepSeek V4、华为昇腾生态 |
| **企业应用场景** | 政务/金融/制造/办公的落地场景足够多，AI公司不需卖到海外也能赚钱 | 产业数字化 |
| **资本退出通道** | 创业公司不能永远靠情怀；需要产业并购+上市+科技基金 | 国内资本市场承接 |

**Manus 被叫停的实质：** 看似监管加强，实则是**倒逼中国构建 AI 产业闭环**。堵住海外退出，倒逼国内资本市场和产业整合接住这些优质 AI 资产。

## 七、产业信号

Manus 被叫停验证了 **AI agent 的战略资产地位**：
- 如果 agent 只是普通应用，监管不会如此重视
- 如果 agent 只是概念炒作，Meta 不会花 20 亿美元收购（对应年收入 1 亿美元）
- Agent 从概念阶段进入战略资产阶段，将代表过去的软件产业

**观察方向：**
1. 明天 AI agent（应用端）是否受资金关注
2. 昇腾生态是否联动
3. 国产算力和 AIDC 是否传导

**中长期：** 关注国内资本市场能否接住、规则是否清晰化。若规则清晰 + 资产接得住，Manus 事件将推动中国 AI 产业完成闭环。
