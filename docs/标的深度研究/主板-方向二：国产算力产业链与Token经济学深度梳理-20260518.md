**第一梯队（必配）**：
1. **长电科技**（先进封装龙头，昇腾核心封测）

### 11.4 关键风险提示

1. **主板缺失AI芯片设计和光模块两大核心环节**，若行情由芯片或光模块引领，主板可能跑输
2. **华为哈勃投资标的全部在科创板**（华丰科技、杰华特、强一股份），主板无法直接配置
3. CANN生态成熟度不及CUDA，第三方库支持有差距
4. 部分标的短期涨幅较大，需警惕情绪退潮后的回调
