## 二、三大观察点（周一）

| 方向 | 验证要点 |
|------|----------|
| **国产算力** | CPU 高开后能否承接，从英特尔概念→国产 CPU/服务器 CPU/分销/整机扩散。超节点/交换芯片/高速互联/整机集成继续回流=科技主线未结束，只是内部高低切 |

## 三、国产算力：午后逆势走强，高低切核心

### 超节点架构

- DeepSeek-V4 正式发布，官方明确下半年**昇腾 950 超节点批量上市**
- 「堆卡」→「网络换算力」范式下，Scale-up 互联环节=增量弹性最大方向
- 整机方案商价值重心从标准化组装→全栈集成迁移

| 标的 | 逻辑 |
|------|------|
| **盛科通信** | 超节点交换芯片核心龙头，国内唯一昇腾生态高端交换芯片独立供应商。25.6Tbps 旗舰芯片批量交付，赛道 26-28 CAGR **231%** |
| **华丰科技** | 华为超节点高速线模组核心供应商，25 年归母 **+2120%**，客户从华为→阿里/浪潮等多家 CSP 扩散 |
| 浪潮信息/中科曙光/华胜天成 | 与华为计算业务共同参与多省市智算中心建设 |
| 泰嘉股份 | 数据中心电源/服务器电源模块代工 |

### CPU 赛道：涨价持续性被低估

- 周五高开低走，但晚间美股英特尔/德州仪器财报超预期单日大涨近 **20%**
- 核心逻辑：Agentic AI 时代 CPU:GPU 配比从 1:4~1:8→**1:1~1:2**，强化学习进一步放大 CPU 战略地位
- Intel 年内三次涨价累计 30%，AMD 最高 15%。先进制程晶圆约束+台积电代工扩产受限→确定性供给短缺

| 标的 | 逻辑 |
|------|------|
| **神州数码** | 英特尔在 A 股唯一代理商，CPU 相关存货约 50~60 亿。同时华为鲲鹏优选合作伙伴+昇腾核心总代理 |
| **禾盛新材** | 控股熠知电子（国产 ARM AI CPU，对标英伟达 Grace），AI 业务 25 年增速 **3952%** |
| 海光信息/寒武纪/龙芯中科 | 国产 CPU 核心 |

### 光模块上游：薄膜铌酸锂

- 3.2T 光模块加速导入→TFLN 高带宽+低驱动电压=下一代主流调制材料。29-31 CAGR **271%**
- **天通股份**：全球少数批量生产铌酸锂晶体，国内唯一 8 英寸铌酸锂晶圆量产
- 光迅科技：TFLN 已有布局
