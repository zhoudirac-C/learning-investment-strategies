# 短线交易阶段全景图

> 情绪周期 · 仓位管理 · 操作策略 | 纪律为先

---

## 第一阶段：情绪反抽点

| 维度 | 内容 |
|------|------|
| **仓位** | 3~5 成 |
| **市场特征** | 题材轮换较快；市场尝试从 2/3/4 板不断试错打开空间；个股独立表现，板块效应偏弱 |
| **操作方向** | 前期老妖股止跌反抽（低吸）；人气股亮眼试错，关注低板弱转强或放量回封（打板）；龙头脱离后关注预期差低吸；龙头见顶后高低切换（排板） |
| **需练习** | 熟悉前期老妖股；打板手法 |

---

## 第二阶段：资金建仓期

| 维度 | 内容 |
|------|------|
| **仓位** | 0~2 成 |
| **市场特征** | 一阶段龙头见顶后的空窗期，调整约一周；主升浪个股缩量调整，洗盘为主 |
| **操作方向** | 规避风险为主，可选空仓；止跌再打板；多关注高位人气股低吸；多分仓 |
| **需练习** | 学会空仓 |

---

## 第三阶段①：主升期·人气形成初期

| 维度 | 内容 |
|------|------|
| **仓位** | 5~7 成 |
| **市场特征** | 板块未完全形成共识，但龙头连续大涨引发关注 |
| **操作方向** | 紧盯新题材（尤其政策方向）；找板块中有人气的首板，参与二板；确认龙头后可提升至 7 成 |
| **需练习** | 识别人气股；学会止损切换 |

---

## 第三阶段②：主升期·人气亢奋期

| 维度 | 内容 |
|------|------|
| **仓位** | 7~10 成 |
| **市场特征** | 板块共识形成，龙头连续涨停；市场深挖补涨 |
| **操作方向** | 重仓参与龙头（低吸/打板）；重仓参与补涨龙（低吸/打板）；半仓打板埋伏人气低位补涨股 |
| **需练习** | 必须参与龙头全过程；牛股首阴战法；学会看板块联动 |

---

## 第三阶段③：主升期·成熟阶段

| 维度 | 内容 |
|------|------|
| **仓位** | 5~8 成 |
| **市场特征** | 龙头高位滞涨，补涨梯队形成；卡位低位新题材更具参与价值；鱼尾阶段需高抛低吸保利润 |
| **操作方向** | 龙头预期差低吸；补涨龙预期差低吸；非主线龙头打板机会 |
| **需练习** | 龙头回头战法 |

---

## 第三阶段④：主升期·衰退阶段

| 维度 | 内容 |
|------|------|
| **仓位** | 2 成 |
| **市场特征** | 龙头与补涨龙走弱，板块大幅下跌；卡位题材出现（多为热点老题材、次新、高送转、公告刺激） |
| **操作方向** | 龙头见顶后高低切换（打板）；人气股反包（多低吸）；新题材人气首板或 2 板机会 |
| **需练习** | 公告板战法 |

---

## 第四阶段：退潮期

| 维度 | 内容 |
|------|------|
| **仓位** | 0~5 成 |
| **市场特征** | 龙头出现见顶信号；短线情绪翻空；个股补跌 |
| **操作方向** | 连续杀跌后，主升阶段龙头、人气股出现低吸机会；其余时间以空仓为主 |
| **需练习** | 龙头回头战法 |

---

> 六个阶段全景：情绪反抽 → 资金建仓 → 人气形成 → 人气亢奋 → 成熟 → 衰退 → 退潮。仓位从 3~5 成逐步升至 7~10 成再降至 0~5 成。纪律为先。
