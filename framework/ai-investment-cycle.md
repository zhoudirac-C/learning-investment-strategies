# AI 算力全产业链炒作路径

## 四阶段模型

博主在 2026 年 4 月 20 日周复盘中明确提出 AI 算力全产业链的四阶段炒作路径。该模型经过多次验证，适用于判断 AI 产业链不同环节的参与时机。

```
第一阶段：算力租赁 → 第二阶段：国产算力芯片 → 第三阶段：AIDC 全产业链 → 第四阶段：CDN
```

## 各阶段特征

### 第一阶段：算力租赁

- **核心逻辑**：Token 经济学确立，Agent 爆发 → Token 消耗激增 → 算力需求爆发
- **关键标的**：能搞到最新最先进卡（GB300）的公司
- **选股标准**：有卡的和没卡的截然不同，必须有实质性利好和产业逻辑
- **参考标的**：协创数据、宏景科技、利通电子、莲花控股

### 阶段 1.5：Token 经济商业化

> 2026 年 5 月新增验证：Token 经济不是独立阶段，而是算力租赁的商业模式升级。

- **核心逻辑**：算力租赁从"按卡/小时计费"进化到"按 Token 效果付费"
- **标志性事件**：
  - 2026-05-16：上海电信推出企业级 Token 算力订阅（1 元=25 万额度点）
  - 2026-05-17：中国电信全国推出 Token 套餐试商用
  - 2026-05-16：宁夏电信 174.38 亿元"Token 工厂"集采（三大运营商首个百亿级 Token 项目）
  - 2026-05-13：百度智能云升级为"Token Factory"
  - 2026-05-14：阿里云 API Token 需求较去年 11-12 月增长超 10 倍
- **市场影响**：运营商凭借骨干网+算力节点优势成为核心组织者，民营算力方向 Token 分成模式升级，市场空间从硬件折旧逻辑切换到智能服务货币化逻辑
- **参考标的**：协创数据（Token 分成模式）、润建股份（五象云谷 2000 亿 Token/小时）、弘信电子（无锡 Token 工厂）

### 第二阶段：国产算力芯片

- **触发条件**：DeepSeek V4 等国产大模型发布，适配国产芯片
- **核心逻辑**：国产替代窗口期被拉长，大厂有钱但买不到卡 → 国产 GPU 受益
- **关键差异**：冲击大模型的公司不太可能全部用国产芯片（与英伟达有差距），国资受力最大
- **参考标的**：沐曦股份、摩尔线程、寒武纪

### 第三阶段：AIDC 全产业链

- **覆盖范围**：PCB、液冷、供电、配电、交换、光模块、存储
- **启动时间**：通常在算力租赁和国产芯片炒作之后，约 5 月中旬左右
- **核心标的**：润泽科技、普冉股份、兆易创新
- **注意**：液冷/供电业绩释放较慢，真正的炒作窗口至少要到下半年

### 第四阶段：CDN

- **核心逻辑**：AI 时代对延迟有新要求，自动驾驶、直播、视频通话、边缘算力都需要 CDN
- **特征**：毛利率从低位开始回升，AI 应用爆发对股价有冲击
- **参考标的**：网宿科技
- **策略**：锁仓不卖，长期看好

## 产业扩散规律

类比上一轮新能源产业链扩散：

```
整车 → 电池 → 锂矿 → 电解液/隔膜 → 铜铝结构件/热管理
```

AI 路径类似：

```
光模块 → 液冷 → 电源 → PCB → 铜箔/电子布/树脂 → AIDC → 超算节点 → 燃气轮机 → 更高端封装材料、光互联 OCS
```

## 2026-05-20 补充：AI硬科技基础设施传导

5月20日复盘从更长周期给出另一条传导路径：

```
AI应用与大模型 → 算力基础设施 → 光通信 → 半导体上游
```

| 阶段 | 驱动 | 主炒方向 |
|------|------|----------|
| AI应用与大模型 | ChatGPT带动应用想象空间 | 办公软件、传媒、游戏、数据要素 |
| 算力基础设施 | 应用催生算力需求 | 服务器、GPU、算力租赁、液冷、电源 |
| 光通信 | 数据中心高速互联需求爆发 | 800G/1.6T光模块 |
| 半导体上游 | 服务器需求放大，产业链瓶颈上移 | 芯片制造、先进封装、存储、CPU/GPU、设备材料 |

该路径与原“四阶段模型”不是冲突关系，而是不同颗粒度：原模型用于交易节奏，本补充用于理解AI硬科技大周期从应用向上游瓶颈迁移。

### 当前环节

当前处于 AI 硬科技从光通信向半导体上游扩散的阶段。行情不再只是炒光模块，而是炒整个 AI 基础设施建设大周期。

### 后续潜在传导方向

1. PCB和覆铜板
2. 液冷和电源
3. 国产CPU/GPU和昇腾产业链
4. 先进封装和检测量测
5. 科技次新

5月21日晚间复盘进一步把国产 CPU、部分 GPU 和软件列为半导体反弹减仓后的潜在承接方向。其定位是科技硬件链内部的高低切和估值修复，不是半导体主线结束后的防御切换。

5月21日收盘复盘进一步把 AI 电源和超级电容列为算力链预期差分支：GB300 起超级电容进入标准电源架构、Rubin 平台储能容量提升，强化 AI 数据中心供电与储能配套升级。该方向仍处新分支验证期，需要观察江海股份等核心标的能否放量突破并带动板块联动。

### 2026-05-21 补充：AI光互连与精密光学

英伟达Rubin平台和AI数据中心升级强化高功耗、高密度、高互连、高散热需求。光模块高位震荡后，资金可能继续在CPO、OCS、光I/O、高速铜缆、液冷、电源、连接器、服务器材料、玻璃基板、高端覆铜板等AI硬件链内部轮动。

光互连进一步把光通信扩展到精密光学元件：

- 微透镜阵列
- FAU
- 滤光片
- 棱镜
- 光栅
- 光学薄膜
- 光学封装
- 微纳结构加工
- 高精度玻璃元件

该方向属于AI基础设施链的扩展分支，需要客户验证和订单兑现，不能只因研报或公告催化直接替代光通信核心资产。

## 操作纪律

1. **当前阶段明确时死磕主线**：如 4 月中下旬死磕算力租赁
2. **阶段切换时提前布局**：5 月中旬从算力切换到 AIDC
3. **不要跨阶段追补涨**：第一阶段没参与的，等第二阶段启动
4. **牛市里好东西横着走**：不会 A 杀下来，给人上车机会
