| 年份 | 观察重点 | 对应问题 |
|------|----------|----------|

## 三、毛利率：定价权的试金石

```
毛利率 = (营业收入 - 营业成本) ÷ 营业收入
```

**毛利率高：** 产品不可替代，有定价权、护城河。
**毛利率低：** 容易被拉进价格战。

- 高科技/高壁垒行业：毛利率需保持在 **30% 以上且稳定上行**
- 制造业代工型：绝对值不重要，看稳定性。10% 但只要多年稳定 + 周转快 + 费用率低，依然可以很好

### 毛利率危险信号

| 类型 | 表现 | 案例 |
|------|------|------|
| **以量换命** | 营收增但毛利持续降 | 隆基绿能 2020-2022 |
| **财务造假型** | 毛利异常攀升 + 应收/存货同步激增 | 航天信息 |
| **虚构繁荣型** | 长期维持超行业高毛利 + 应收/存货激增 | 可能直接退市 |
| **渠道压货型** | 向渠道超量铺货，高毛利 + 应收飙升 | 医药/系统集成公司 |
| **资产迷雾型** | 生物资产/水产养殖等难以审计的资产 | 獐子岛类 |

**核心：毛利率出现异常 + 应收/存货背离 = 危险信号。**

**正面案例：中际旭创。** 2023 年前毛利 < 30%，2023 年后突破 30% 进入高毛利区间 → 股价同步暴涨，形成戴维斯双击。

## 四、净利润与净利率：真正赚了多少

毛利只是差价，扣完租金、人工、研发、水电之后剩下的才是净利润。

### 三个核心问题

1. **规模越大越赚钱，还是越做越累？** 健康公司：规模效应，净利率逐步改善。问题公司：营收涨，费用同步甚至更快涨，净利没增长。

2. **利润有多少来自主营业务？** 卖资产、拿补贴、炒股收益 → 不可持续。真正的价值来自**经常性收益**。

3. **费用结构是否健康？** 三费（销售/管理/研发）+ 财务费用（利息支出）+ 非经常性损益占比。

### 四类必盯数据

| 指标 | 含义 |
|------|------|
| 净利润/净利率 | 最终盈利水平和效率 |
| 三费及费用率 | 赚钱的消耗 |
| 财务费用（利息支出） | 是否为杠杆付出越来越高代价 |
| 非经常性损益占比 | 有多少利润来自一次性因素 |

### 正面 vs 反面

| 公司 | 特征 |
|------|------|
| **宁德时代** | 营收快速增长 + 净利率稳定增长，利润全部来自主营业务 |
| **立讯精密** | 增长进入平台期，两头受压（大客户议价+成本刚性上涨），净利率低位徘徊 |
| **京东方 A** | 强周期性：好时三十几的毛利，差时急剧下滑，巨额亏损；每年必须高研发投入维持竞争力 |

**规模扩大 ≠ 盈利能力增强。**

## 五、净利润现金含量：账上利润是真的吗

```
净利润现金含量 = 经营活动现金流净额 ÷ 归母净利润
```

**> 0 且稳定 → 真金白银在挣钱。长期为负 → 赊销或压货维持表面增长。**

### 现金是企业的生命线

卖货是收现金还是赊账？应收账款多少？存货多少？

**案例：金域医学。** 疫情期间报表极漂亮，但资产负债表里应收账款极大 → 表面安全实则积累风险，收紧后集中爆发。

## 六、营收同比增长：增长的质量

### 三层追问

1. **增长从哪里来？** 老客复购（商业模式扎实）vs 新客增长（市场仍有空间）vs 不断追逐新渠道（可持续性差）
2. **靠提价还是放量？** 提价 = 品牌力支撑；放量 = 自然需求 vs 投流/补贴砸出来（需谨慎）
3. **增长有没有拖垮毛利/净利？** 暂时性投资 vs 结构性价格战

### 健康增长的两个条件

1. 增长可持续，不是昙花一现
2. 增长质量好，毛利和净利能守得住

### 中际旭创案例：辩证看应收与存货

- 核心客户英伟达，付款提货节奏直接影响财务表现
- 应收 + 存货同步攀升 → 需分析：
  - 应收增速 vs 营收增速：若应收快于营收，回款效率下降
  - 合同负债是否同步增长：若是，存货增加更多来自订单驱动，风险可控
  - 同行业对比（新易盛）：若仅此一家独有，可能是大客户谈判地位所致

## 七、货币资金：账上真有钱吗

### 三类现金，三种意义

| 来源 | 含义 |
|------|------|
| 经营赚来的 | 商业模式有了自我造血能力 |
| 借来的 | 用时间换空间，同时积累风险 |
| 未使用的投资款 | 暂时缓冲，不代表经营能力 |

**理想状态：** 现金随经营改善逐渐增加，能覆盖未来 6-12 个月固定支出。

### 现金迷雾

**万科案例：** 账上巨额现金，但有大量受监管的预售资金/项目专款无法动用，同时背负巨额有息负债 → 财务费用持续侵蚀利润。

**高现金 + 高负债 + 高利息的陷阱：** 通过借款成本购买财务报表上稳健的形象。

**穿透方法：** 计算净现金 = 货币资金 - 所有有息负债。结果为巨额负数 = 现金充裕是幻觉。

## 八、负债与偿债能力

### 关键问题

- 借钱是为了兑现确定的订单，还是赌未来增长？
- 借的是短期还是长期？（死因往往是负债来得太急、到期太快，而非负债太多）

### 核心指标

| 指标 | 公式 | 意义 |
|------|------|------|
| **资产负债率** | 总负债 ÷ 总资产 | 整体杠杆水平 |
| **有息负债规模** | 短期借款 + 长期借款 + 应付债券 | 对现金流的刚性要求 |
| **流动比率** | 流动资产 ÷ 流动负债 | > 1 是生死线，> 2 是安全垫 |
| **速动比率** | (流动资产 - 存货) ÷ 流动负债 | > 1 安全，剔除存货后仍能偿债 |

### 长江电力：特例

流动比率和速动比率极低，但不是流动性紧张——电站计入固定资产（非流动资产），建设融资是长期借款（非流动负债）。业务根本不需要囤积大量短期运营资金。**本质是现金奶牛。**

### 阳光电源：正面案例

速动比率 > 1，流动比率 > 1 → 短期偿债能力优秀，最难那几个月也熬得过去。

### 牧原股份：典型反面

猪周期好时赚大钱 → 盲目扩张 → 短期借款急剧攀升 → 猪周期反转 → 巨亏。

### 危险信号

- 资产负债率持续上升 + 经营现金流不稳定甚至走弱 → 用负债对冲经营问题
- 短期负债过高 + 银行收缩授信 → 流动性断裂

## 九、产权比例：杠杆的尺度

```
产权比例 = 总负债 ÷ 股东权益
```

**股东每投入 1 元，借了多少元来扩大生意。**

- **持续下降：** 通过盈利消化债务，财务根基日渐趋稳
- **持续攀升：** 不断加码杠杆，还款压力越来越高

| 产权比例 | 判断 |
|----------|------|
| < 1.2 | 相对稳妥 |
| 逐年下降 | 积极信号，摆脱对杠杆的依赖 |

## 十、应收账款与存货周转

### 应收账款周转

```
周转率 = 营业收入 ÷ 平均应收账款
周转天数 = 365 ÷ 周转率
```

- 周转天数越短越好，至少不要逐年拉长
- 应收增速不应长期跑赢营收增速
- 营收增但应收增速更快 = 通过赊账换增长

### 存货周转

```
周转率 = 营业成本 ÷ 平均存货
```

**存货 = 钱压在仓库里。**

- 畅销品库存 = 保障交付
- 滞销品库存 = 已经是损失

**最危险的组合信号：** 存货增长长期高于营收增长 + 周转天数不断拉长 → 滞销风险 → 被迫打折（毛利崩塌）→ 计提跌价准备（侵蚀利润和净资产）

## 十一、完整分析框架

把以上所有指标套在一起看：

```
利润真实性 → 偿债能力 → 增长能力 → 运营效率
    ↓            ↓           ↓          ↓
毛利率/净利率  流动/速动比  营收同比  应收/存货周转
现金含量      产权比例    增长来源  周转天数趋势
             有息负债     毛利净利联动
```

**核心：抓住三张表的关系，用创业的框架去理解，就能看懂一家公司。**
