## 八、技术信号：指标使用边界（早盘动态补充）

> "不要在个股上用背离指标。现在大多数指标只能用在大市值和指数上，小票不能用。"

### 核心观点

- **背离指标不适用个股**：MACD顶背离、底背离等技术指标在个股层面的可靠性极低
- **指标有效性分层**：
  - ✅ **大市值标的**：指标有一定参考价值，因机构资金主导、波动相对有序
  - ✅ **指数层面**：指标有效性最高，因综合反映市场整体状态
  - ❌ **小票/题材票**：指标基本失效，因游资主导、波动剧烈、易受消息和情绪驱动

### 原因分析

小票不能用技术指标的核心原因：

1. **资金性质不同**：小票多由游资和散户主导，进出节奏快、无规律，技术指标基于历史价格和成交量计算，无法反映突发性资金行为
2. **操纵空间大**：小市值标的易被短期资金影响走势，K线形态和指标信号可能被人为制造
3. **流动性差异**：小票买卖盘薄，单笔大单即可改变价格，指标信号噪音远大于有效信号
4. **机构参与度低**：大市值标的有机构资金持续跟踪，行为相对可预测；小票缺乏这种"稳定力量"

### 实践建议

| 标的类型 | 指标使用建议 |
| --- | --- |
| 大盘指数（沪指/深成指/创业板指） | 技术指标有效，可作为判断依据 |
| 大市值核心蓝筹（市值>500亿） | 指标有一定参考价值，需结合基本面 |
| 中小市值（市值<200亿） | 指标基本失效，不要依赖背离信号做决策 |
| 题材/概念小票 | 完全不看技术指标，专注题材催化和资金流向 |
