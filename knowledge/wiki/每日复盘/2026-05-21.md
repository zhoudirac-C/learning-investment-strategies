# 2026-05-21 复盘：情绪分歧日与科技内部轮动

## 核心定性

5月21日属于反弹途中的情绪分歧日，而不是主线直接退潮。盘面主轴仍在科技内部，但从半导体核心扩散到京东方系、玻璃基板、CPU、无人驾驶等低位方向，说明风险偏好仍在，只是短线一致性被打散。

当日最重要的变化是：半导体设备从连续逼空转入分歧洗盘，Micro-LED/玻璃基板从强催化转入承接验证。两者同属科技链，但交易属性不同，不能用同一套容错率处理。

## 早盘观察框架

早盘核心定调是反弹途中不追高，持股做T打利润垫，聚焦AI与存储主线内部轮动。

三大节奏观察：

| 观察项 | 健康信号 | 风险信号 |
|--------|----------|----------|
| 开盘幅度 | AI核心小幅高开或低开快速修复 | AI核心大幅高开后放量回落 |
| 量能健康度 | 放量能推动指数和核心方向 | 早盘大幅放量但指数无法上推 |
| 科创50强度 | 科创强且三大指数同步修复 | 科创继续强于主板但全A分化 |

AI硬件重点看“易中天”：新易盛、中际旭创、天孚通信。英伟达财报超预期后，交易重点从财报好坏切换到承接质量和利好兑现。

存储方向由长鑫科技上会和长江存储辅导形成双催化。最佳结构不是全线高潮，而是核心中军稳住，半导体设备、材料、封装材料继续轮动补涨。

## 盘中主线分化

半导体主线没有明显情绪核心，但北方华创、中微公司等趋势容量票稳定，中芯国际、兆易创新等中军震荡承接，说明核心趋势没有被破坏。

京东方与康宁合作带动京东方系、玻璃基板和Micro-LED发酵。该方向的优势是低位、强催化、产业链联动，问题是合作备忘录不是确定性订单，短线更像主线外溢后的题材补涨。

券商和医药的强势暂时按低位轮动修复处理。判断它们是否干扰科技，关键看券商拉升时科技核心是否同步跳水。若科技核心只是震荡承接，则不构成主线切换。

## 午盘定性

午盘确认当日是情绪分歧日。半导体设备分歧不等于主线结束，更像连续上涨后的正常洗盘。这个方向背后有景气度、订单、国产替代、长鑫/长存产业催化和机构趋势资金共识，抗分歧能力强于普通题材。

Micro-LED被砸得较重，原因在于它更偏远期叙事和资金接力，缺少半导体设备那种订单、业绩和产业景气度支撑。遇到AI和半导体拥挤度小高潮后的兑现时，开盘追高资金容易被动。

## 操作纪律

当日不是简单全卖或全留，而是按趋势状态分层：

| 类型 | 处理 |
|------|------|
| 破趋势方向 | 反抽按弱修复处理，不能恋战 |
| 趋势还在且逻辑硬 | 可以多等一等，看分歧后是否回流 |
| 远期题材补涨 | 只看承接，不用主线容错率保护 |
| 已有利润核心票 | 做T打利润垫，避免坐过山车 |

午后仓位策略是半仓轮动做T，把仓位降到舒服位置，剔除破趋势方向，保留趋势还在、逻辑还硬的方向。

尾盘低吸只适用于全天单边下跌后的2点55附近；如果盘中已经反弹，则放弃，不做反弹追高。

## 晚间视频复盘上修

晚间视频复盘对指数结构做了更大级别修正：此前 ABC 下杀的 C 浪目标在 4080 点附近，指数已一步到位打到 4077-4082 区间，因此不再按常规 ABC 下杀处理，而重新界定为整体 A 浪下杀。新大周期反弹目标从 4200 上修到 4700。

短线节奏：

| 节点 | 含义 |
|------|------|
| 4047 | 47 日线支撑，后续回踩观察位 |
| 4170 | 若三天内反弹到位，5月27日顺势减仓 |
| 5月27日 | 长鑫科技 IPO 上会，半导体减仓规避节点 |
| 长鑫正式上市 | 半导体阶段见顶信号，需全部撤退 |

晚间对半导体的态度比午盘更积极：半导体行情尚未结束，5月21日尾盘低吸或加仓方向可继续持有，但需要围绕 5月27日上会和正式上市两个节点管理仓位。

后续潜在承接方向从半导体扩展到国产 CPU、部分 GPU 和软件；光互联、微透镜作为新方向可看修复，但只看一线光板块。机器人被降级为调整期轮动题材，不适合重仓。

## 收盘追加：真恐慌但非崩盘

另一篇 5月21日复盘把当天盘面定义为“真恐慌但非崩盘”。两市 4773 只个股下跌，跌幅超过 7% 的个股达到 395 只，半导体、芯片、CPO 等科技方向跌幅均超过 4%，恐慌是真实的。

但成交额放大到约 3.5 万亿，接近节后最高量能，同时封死跌停没有大面积扩散，说明并非只有卖盘而没有承接。更准确的定性是：市场从单边逼空进入高位宽幅震荡和去弱留强阶段。

新的操作框架：

| 维度 | 判断 |
|------|------|
| 市场状态 | 高位放量分歧后的筹码清洗，不是直接崩盘 |
| 科技主线 | 半导体、芯片、CPO 杀跌但逻辑未破坏，进入龙头筛选 |
| A档活口 | PCB、先进封装 |
| 潜在承接 | 机器人、具身智能 |
| 新预期差 | AI电源、超级电容 |
| 防御轮动 | 食品饮料、储能只按过渡防御，不看主线 |

次日最优剧本是低开急杀后快速拉回。如果开盘下探 1%-2%，10 点前出现放量拉升，并且机器人或 PCB 率先翻红，可用 1-2 成仓位低吸强势方向龙头。若低开后弱反弹、午后继续回落，则不抄底，弱势股反弹减仓。若直接高开反弹，则先观望，防止高开缩量冲高回落。

仓位维持半仓至六成，不满仓也不清仓。持仓按强势、中性、弱势三类处理：趋势未破且量能承接好的继续持有或做 T；跟随下跌但未破位的看反弹力度；放量破位且反弹无力的去弱换强。
