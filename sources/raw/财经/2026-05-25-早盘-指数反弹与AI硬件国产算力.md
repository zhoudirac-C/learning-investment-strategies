## 二、指数研判

### 1. 技术面

日线顶部结构形成后，市场已经连续调整 6 个交易日。

上周五三大指数高开高走，创业板指、深成指均涨超 2%，但上证指数未能反包周四阴线，显示上方压力依然存在。

### 2. 今日变量

地缘协议能否达成是今日最大变量。

如果协议达成，市场可能将其视为阶段性缓和，指数先冲高；但由于短期反弹级别较小，需要防范冲高回落。

如果指数能高举高打，并用两天上涨抵消周四调整，则说明反弹强度超预期。

### 3. 后续展望

本周三前继续看反弹，重点观察成交量配合情况。

比较现实的走势是类似去年 10 月：先缩量整理，再回踩确认，之后继续向上。

### 4. 观察与证伪

| 观察项 | 偏强信号 | 风险信号 |
| --- | --- | --- |
| 上证指数 | 放量突破 5 日均线 | 高开后缩量回落，无法站稳 5 日均线 |
| 反弹节奏 | 两天上涨抵消周四调整 | 仅小级别 30 分钟反弹后再度转弱 |
| 成交量 | 指数上涨伴随有效放量 | 反弹缩量、热点轮动过快 |
| 情绪 | 一进二晋级率按 15% 计算达到 15 家及以上 | 连板接力弱，强势股冲高回落 |

## 三、板块与题材机会

### 1. AI硬件产业链：核心主线

#### 英伟达 Rubin 服务器催化

用户笔记引用的摩根士丹利拆解显示，Rubin / VR200 机柜售价约 780 万美元，较 GB300 的 399 万美元增长近 100%。

零部件价值量增幅：

| 环节 | 价值增幅 |
| --- | ---: |
| PCB | +233% |
| MLCC | +182% |
| ABF 基板 | +82% |
| 电源 | +32% |
| 液冷组件 | +12% |

#### PCB价值提升逻辑

PCB 价值提升来自层数升级和新增模块：

- 新增 ConnectX 模块 PCB；
- 新增 44 层中板 PCB；
- 原有计算板从 22 层升级至 26 层；
- 交换托盘从 24 层升级至 32 层。

#### MLCC价值提升逻辑

MLCC 单柜使用量约 60 万个，比 GB300 高出 30% 以上。

用量提升主要来自：

- 计算板用量提升；
- 交换板用量提升；
- 新模块带来额外需求。

#### 重点关注方向

| 方向 | 关注标的 |
| --- | --- |
| PCB / CCL | 鹏鼎控股、深南电路、胜宏科技、华正新材 |
| MLCC | 风华高科、三环集团、法拉电子、昀冢科技 |
| ABF 载板 | 方邦股份、兴森科技 |
| 玻璃基板 | 京东方 A、南玻 A |

玻璃基板方向中，京东方 A、南玻 A 上周五竞价被错杀，今日需要观察能否主动走强。

#### 盘面验证

| 观察项 | 意义 |
| --- | --- |
| 鹏鼎控股能否继续封板 | 验证 PCB 方向强度 |
| 深南电路、胜宏科技等千亿市值公司能否保持强势 | 验证大市值趋势资金承接 |
| 20cm 标的表现 | 验证题材弹性与短线风险偏好 |
| 京东方 A、南玻 A 能否主动走强 | 验证玻璃基板是否从错杀修复转为主动进攻 |

#### DeepSeek永久降价催化

用户笔记提到，DeepSeek-V4-Pro API 永久降价至原价的 1/4，每百万 tokens 输入（缓存命中）0.025 元，创全球大模型价格新低。

该催化的推演逻辑是：降价有望利好应用开发和 AI 普惠，并强化国产模型与国产算力的闭环。

## 五、今日操作策略与关键观察点

### 1. 操作策略

仓位建议控制在 5-7 成，避免追高。

优先低吸调整充分的 AI 硬件核心标的，重点关注上周五领涨的 PCB / MLCC 方向持续性，尤其是 20cm 标的表现。

如果指数冲高回落，可考虑减仓部分短期涨幅过大的品种，等待回踩机会。

> 注：以上为原始笔记中的策略表达，不构成无条件买卖指令。实际执行需结合盘中量能、板块持续性、个股趋势位置与个人持仓风险。

### 2. 关键观察点

| 序号 | 观察点 | 观察意义 |
| ---: | --- | --- |
| 1 | 一进二晋级率 | 按 15% 计算，需要在 15 家及以上，反映市场情绪强度 |
| 2 | 成交量 | 上证指数能否放量突破 5 日均线，是反弹能否延续的关键 |
| 3 | AI硬件板块 | 鹏鼎控股能否继续封板，深南电路、胜宏科技等千亿市值公司能否保持强势 |
| 4 | 玻璃基板方向 | 京东方 A、南玻 A 能否主动走强，消除市场对反弹持续性的疑虑 |
| 5 | 国产算力板块 | 超聚变相关概念股能否与英伟达海外链共同走强，形成良性互动 |

### 3. 触发条件与失效条件

| 场景 | 触发条件 | 对应处理 |
| --- | --- | --- |
| 反弹超预期 | 指数高举高打，放量突破 5 日均线，AI硬件与国产算力共振 | 维持 5-7 成观察，优先跟踪核心趋势与主动走强标的 |
| 弱反弹 | 指数高开后缩量横盘，题材分化但核心标的不破趋势 | 不追高，等分歧低吸或回踩确认 |
| 冲高回落 | 指数高开低走，强势股炸板，成交量不能配合 | 降低短线涨幅过大品种暴露，等待回踩机会 |
| 主线证伪 | PCB / MLCC 核心股不持续，国产算力只剩映射炒作，玻璃基板不能主动修复 | 降低对 AI 硬件短线持续性的预期，观察机器人、电力等轮动方向 |
