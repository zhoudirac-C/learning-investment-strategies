# 主线判断方法论

## 核心标准

只有**成板块建制的进攻**，才具备有序性和持续性。局部个股脉冲难以形成真正的主线级别进攻。

## 板块阵型要素

一个具备主线潜质的板块需要具备以下要素：

1. **前排涨停股**：至少有若干只涨停或强势封板，形成赚钱效应
2. **容量中军**：有一只或几只大市值、高成交的核心标的作为板块中军，提供安全边际
3. **后排跟风**：板块内有多个标的联动上涨，形成梯队
4. **产业逻辑**：有明确的产业消息或逻辑支撑，而非单纯的情绪炒作

## 新生启动 vs 阶段新主流

| 维度 | 新生启动 | 阶段新主流 |
|------|---------|-----------|
| 指数共振 | 无或短暂 | 有，持续 |
| 反包结构 | 缺少"反包护身符" | 具备完整反包结构 |
| 持续性 | 待观察 | 已验证 |
| 参与策略 | 控制仓位观察 | 可适当加仓参与 |

## 泛科技修复中的主线筛选

泛科技修复不等于单一主线启动。若机器人、存储、光通信、算力租赁等方向一起轮动，但没有一个方向真正带队，仍按修复行情处理。

二波启动需要同时看到：

1. 指数弱分歧不破位
2. 量能继续放大
3. 昨日大长腿核心不补跌
4. 低位新分支继续晋级
5. 科技内部出现明确主线，如国产算力、存储链、半导体设备材料、光通信之一形成持续聚焦

局部强势的板块，还要继续观察能否扩散成板块级持续进攻。长鑫链条若只有兆易创新和少数设备股强，而核心设备、材料、零部件分化明显，则不能直接定义为全面强一致。

## 主线强度与全市场共振

主线强，不等于市场强。科创50和半导体强势时，需要区分“带动型上涨”和“虹吸型上涨”。

健康的主线接棒应同时满足：

1. 容量中军持续走强
2. 设备、材料、封测、国产算力等梯队扩散
3. 上证和创业板同步修复
4. 全 A 上涨家数明显回升
5. 金融、周期、消费等非科技方向至少不明显失血

如果只有少数科创权重和半导体核心上涨，全 A 大量下跌，则是局部主线虹吸，不是全市场共振。

## 主线外溢 vs 主线否定

主线强势后，资金可能从核心方向外溢到同一大产业链里的低位分支。只要这种扩散仍发生在科技内部，就不能简单理解为主线退潮。

| 现象 | 判断 |
|------|------|
| 半导体核心稳定，资金扩散到京东方系、CPU、无人驾驶等科技低位 | 科技内部高低切，风险偏好仍在 |
| 券商、医药拉升，但半导体、存储、AI硬件核心不跳水 | 低位宽度修复，不构成主线切换 |
| 券商拉升时科技核心同步跳水 | 可能出现资金抽离科技，需要警惕 |
| 远期题材一字高潮后分歧无承接 | 主线外溢补涨失败，不能代表主线本身结束 |

主线与补涨分支要分开估值。半导体设备有景气度、订单和扩产支撑，Micro-LED等远期题材更多依赖资金接力和预期，抗分歧能力天然不同。

## 板块普涨后的龙头筛选

当主线经历连续逼空后出现高位放量大分歧，不能再沿用“板块都涨”的思路。此时主线可能没有结束，但交易机会会从板块性机会切换成个股筛选机会。

优先保留三类：

1. 趋势完整的核心龙头。
2. 产业逻辑继续强化的细分方向。
3. 补跌后能快速反包的强势股。

坚决降低三类预期：

1. 短期涨幅过大、筹码松动的高位跟风。
2. 放量破位后反弹无力的弱势股。
3. 只靠低位和故事驱动、没有订单或业绩验证的补涨股。

强分歧后要看谁主动承接资金：若机器人、PCB、先进封装、AI电源等科技内部方向率先翻红放量，说明资金仍在科技内部做结构切换；若只剩食品饮料、储能等低位防御修复，则不能定义为主线修复。

## 周末产业逻辑复盘

强分歧后的修复窗口，不只看短线涨跌，还要用产业逻辑重新校准持仓和主线层级。

周末复盘重点：

1. 当前持仓个股的强弱。
2. 科技主线内部的产业逻辑是否继续强化。
3. 近期资金偏好是否发生变化。
4. 哪些方向是趋势核心。
5. 哪些方向只是短线轮动或支线补涨。

如果后续视频更新个股和产业逻辑，应优先用来校准“趋势核心 vs 短线轮动”，而不是只记录单个标的涨跌。

## 技术催化与风格切换的关系

5月25日盘中补发提供了一个主线判断样本：半导体上涨和周五 PCB 上涨本质都是技术催化，但催化只解决“为什么能加速”，不能直接解决“能不能持续”。

判断时要拆成三层：

| 层级 | 问题 | 5月25日案例 |
|------|------|-------------|
| 产业催化 | 是否有新技术、新架构、新平台 | Rubin 对 PCB，华为新技术对半导体/先进封装 |
| 交易位置 | 原本是否已经拥挤或需要调整 | 半导体一致性过高，叠加批量减持 |
| 承接验证 | 次日能否延续，而不是高开低走 | PCB 当天已高开低走，半导体需看次日 |

如果资金同时开始流向老登股，说明市场可能在做风格切换。此时科技主线未必立刻结束，但主线判断要从“产业逻辑强”升级为“产业逻辑强 + 资金承接强”。缺少承接时，优先保护利润，不把远期技术叙事当作追高理由。
