# 个股分析 Playbook

1. 确认标的身份、代码、市场。
2. 获取真实行情、K 线、分时、资金、财务和新闻。
3. 检索本地 claims/wiki/cases 中的标的、板块和主线。
4. 判断市场周期和板块阶段。
   - 周期先分为主升、混沌轮动、强分歧、退潮、冰点、修复。
   - 主升未破坏时以持核心为主；混沌轮动时买阴卖阳；退潮时降低仓位等冰点。
   - 若处于高位强分歧后，先检查指数关键位、量能、跌停家数和主线回流。
   - 防御方向护盘不等于进攻修复，必须确认主线核心是否主动承接。
   - 分歧初期的逆势活口要区分退潮活口和复苏活口。
   - 若科技主线处于高位，必须检查融资买入、换手率和成交占比；拥挤度过高时，逻辑未结束也要先按震荡消化处理。
   - 高估值科技方向要纳入宏观压力测试：10 年期美债、油价、美元、人民币和 PPI 修复。
   - 顶部钝化、高 9、高 10 是风控信号；若钝化消失，需要按规则纠错。
5. 判断个股是核心、跟风、补涨、案例还是过期标的。
   - 优先买核心回调，避免追后排上涨。
   - 主线完成全链条补涨后，后排跟风风险显著上升。
6. 执行 F10 基本面分析。
   - 财务估值需结合市场量能；高量能提高成长估值容忍度，缩量和退潮时收紧标准。
   - 结构性再通胀阶段，要区分周期涨价链、AI 上游材料和高端制造电力设备，不把所有涨价方向等同处理。
7. 区分证据、解释、推断。
8. 输出证伪条件、下一期跟踪字段和风险提示。
